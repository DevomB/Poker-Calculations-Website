# `expectedValueRaiseWithRake`

Raise EV with rake on called pot.

<div className="apiSignature"><code>expectedValueRaiseWithRake(equityWhenCalled: number, potBeforeRaise: number, raiseSize: number, foldEquity: number, potIfCalled: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Same as [`expectedValueRaise`](/docs/reference/api/pot-and-ev/expected-value-raise) but rake is taken from `potIfCalled` when called.

## How to use

```js
const ev = poker.expectedValueRaiseWithRake(
  0.45, 60, 45, 0.35, 150,
  0.05, 3,
);
const evNoRake = poker.expectedValueRaise(0.45, 60, 45, 0.35, 150);
console.log({ ev, evNoRake }); // ev <= evNoRake
```
