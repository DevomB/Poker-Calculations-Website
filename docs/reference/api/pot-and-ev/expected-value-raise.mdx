# `expectedValueRaise`

Chip EV of raise vs fold (two-outcome model).

<div className="apiSignature"><code>expectedValueRaise(equityWhenCalled: number, potBeforeRaise: number, raiseSize: number, foldEquity: number, potIfCalled: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';

```

## When to use

Use when you need chip ev of raise vs fold (two-outcome model) in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.expectedValueRaise(/* see signature above */);
console.log(result);
```
