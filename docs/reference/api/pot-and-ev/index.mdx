# Pot & EV

Chip-level pot odds, breakeven equity, call EV, rake adjustments, and book-style winning odds. These helpers assume a single call decision on the current street—no future-street modeling beyond what rake on the final pot implies.

Use this group when you are building call/fold trainers, HUD overlays, or server-side spot analysis where you already have equity and need to compare it to the price of a call.

| Export | Summary |
| --- | --- |
| [`potOddsRatio`](/docs/reference/api/pot-and-ev/pot-odds-ratio) | `toCall / (pot + toCall)` when valid; else `0`. |
| [`expectedValueCall`](/docs/reference/api/pot-and-ev/expected-value-call) | Chip EV of calling once vs folding (0); no future streets. |
| [`expectedValueCallWithRake`](/docs/reference/api/pot-and-ev/expected-value-call-with-rake) | Call EV when the final HU pot pays rake. |
| [`breakevenCallEquity`](/docs/reference/api/pot-and-ev/breakeven-call-equity) | Same fraction as `potOddsRatio` for chip calls. |
| [`breakevenCallEquityFromPotOddsDisplayRatio`](/docs/reference/api/pot-and-ev/breakeven-call-equity-from-pot-odds-display-ratio) | `1/(1+R)` from display ratio `R`; `0` when `R` is `+∞`. |
| [`potOddsDisplayRatioFromBreakevenCallEquity`](/docs/reference/api/pot-and-ev/pot-odds-display-ratio-from-breakeven-call-equity) | Inverse `(1−e)/e`; `+∞` when `e=0`; `0` when `e=1`. |
| [`formatPotOddsReducedFraction`](/docs/reference/api/pot-and-ev/format-pot-odds-reduced-fraction) | Reduced integer `pot : to_call` string. |
| [`equityToWinningOddsAgainst`](/docs/reference/api/pot-and-ev/equity-to-winning-odds-against) | Book-style `(1−p)/p`; `+∞` at `p=0`. |
| [`winningOddsAgainstToEquity`](/docs/reference/api/pot-and-ev/winning-odds-against-to-equity) | `1/(1+o)` for `o≥0`; `0` when `o=+∞`. |
| [`rakeFromPot`](/docs/reference/api/pot-and-ev/rake-from-pot) | Rake `min(fraction×pot, cap)`. |
| [`breakevenCallEquityWithRake`](/docs/reference/api/pot-and-ev/breakeven-call-equity-with-rake) | Breakeven equity when the final pot pays rake. |

Display-oriented pot odds (`potOddsRatioDisplay`, `formatPotOdds`) live under [Stacks & display](/docs/reference/api/stacks-and-display/).
| [`netPotAfterRake`](/docs/reference/api/pot-and-ev/net-pot-after-rake) | Pot minus rake: `pot − rakeFromPot`. |
| [`netPotAfterCallAndRake`](/docs/reference/api/pot-and-ev/net-pot-after-call-and-rake) | Net pot after HU call and rake. |
| [`effectivePotOddsDisplayAfterRake`](/docs/reference/api/pot-and-ev/effective-pot-odds-display-after-rake) | Display pot:call after rake on final pot. |
| [`expectedValueRaise`](/docs/reference/api/pot-and-ev/expected-value-raise) | Chip EV of raise vs fold (two-outcome model). |
| [`expectedValueRaiseWithRake`](/docs/reference/api/pot-and-ev/expected-value-raise-with-rake) | Raise EV with rake on called pot. |
| [`breakevenRaiseEquity`](/docs/reference/api/pot-and-ev/breakeven-raise-equity) | Equity needed for neutral raise. |
| [`breakevenCallEquityWithPostedAnte`](/docs/reference/api/pot-and-ev/breakeven-call-equity-with-posted-ante) | Breakeven when hero still posts ante. |
| [`potSizeAfterHuCall`](/docs/reference/api/pot-and-ev/pot-size-after-hu-call) | Pot after heads-up call: `pot+2×call`. |
| [`potSizeAfterHuBet`](/docs/reference/api/pot-and-ev/pot-size-after-hu-bet) | Pot after heads-up bet: `pot+2×bet`. |
| [`expectedValuePerBigBlind`](/docs/reference/api/pot-and-ev/expected-value-per-big-blind) | Chip EV expressed in big blinds. |
