# `potSizeAfterHuCall`

Pot after heads-up call: `pot+2×call`.

<div className="apiSignature"><code>potSizeAfterHuCall(potBeforeCall: number, toCall: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Use when you need pot after heads-up call: `pot+2×call` in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.potSizeAfterHuCall(/* see signature above */);
console.log(result);
```
