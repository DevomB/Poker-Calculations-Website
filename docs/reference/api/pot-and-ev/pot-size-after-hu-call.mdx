# `potSizeAfterHuCall`

Pot after heads-up call: `pot+2×call`.

<div className="apiSignature"><code>potSizeAfterHuCall(potBeforeCall: number, toCall: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Chip pot after a heads-up call: `potBefore + 2×toCall`. Same total used inside [`sprAfterCall`](/docs/reference/api/sizing-and-commitment/spr-after-call).

## How to use

```js
const potBeforeCall = 80;
const toCall = 40;

const newPot = poker.potSizeAfterHuCall(potBeforeCall, toCall);
console.log(newPot); // 160
```
