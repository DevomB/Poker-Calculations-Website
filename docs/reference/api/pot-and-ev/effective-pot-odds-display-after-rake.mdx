# `effectivePotOddsDisplayAfterRake`

Display pot:call after rake on final pot.

<div className="apiSignature"><code>effectivePotOddsDisplayAfterRake(potBeforeCall: number, toCall: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Display-style **net pot : call** after rake on the final pot. Inverse of breakeven equity with rake.

## How to use

```js
const potBeforeCall = 100;
const toCall = 50;
const rakeFraction = 0.05;
const rakeCap = 5;

const display = poker.effectivePotOddsDisplayAfterRake(
  potBeforeCall,
  toCall,
  rakeFraction,
  rakeCap,
);
const need = poker.breakevenCallEquityWithRake(potBeforeCall, toCall, rakeFraction, rakeCap);
console.log(display, 1 / need); // display ≈ 1/need
```
