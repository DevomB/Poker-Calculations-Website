# `potSizeAfterHuBet`

Pot after heads-up bet: `pot+2×bet`.

<div className="apiSignature"><code>potSizeAfterHuBet(potBeforeBet: number, betSize: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Chip pot after a heads-up bet that is called: `potBefore + 2×betSize`.

## How to use

```js
const potBeforeBet = 60;
const betSize = 40;

console.log(poker.potSizeAfterHuBet(potBeforeBet, betSize)); // 140
```
