# `potSizeAfterHuBet`

Pot after heads-up bet: `pot+2×bet`.

<div className="apiSignature"><code>potSizeAfterHuBet(potBeforeBet: number, betSize: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);

```

## When to use

Use when you need pot after heads-up bet: `pot+2×bet` in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.potSizeAfterHuBet(/* see signature above */);
console.log(result);
```
