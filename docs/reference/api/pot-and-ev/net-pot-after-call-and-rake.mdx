# `netPotAfterCallAndRake`

Net pot after HU call and rake.

<div className="apiSignature"><code>netPotAfterCallAndRake(potBeforeCall: number, toCall: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Final pot after a heads-up call (`potBefore + 2×toCall`), then rake applied. Matches the pot size inside [`breakevenCallEquityWithRake`](/docs/reference/api/pot-and-ev/breakeven-call-equity-with-rake).

## How to use

```js
const potBeforeCall = 90;
const toCall = 30;
const rakeFraction = 0.05;
const rakeCap = 3;

const net = poker.netPotAfterCallAndRake(potBeforeCall, toCall, rakeFraction, rakeCap);
const finalPot = potBeforeCall + 2 * toCall;
console.log({ finalPot, net });
```
