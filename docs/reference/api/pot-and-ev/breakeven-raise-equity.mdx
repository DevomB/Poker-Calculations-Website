# `breakevenRaiseEquity`

Equity needed for neutral raise.

<div className="apiSignature"><code>breakevenRaiseEquity(potBeforeRaise: number, raiseSize: number, foldEquity: number, potIfCalled: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Equity-when-called that makes [`expectedValueRaise`](/docs/reference/api/pot-and-ev/expected-value-raise) exactly zero for fixed sizing and fold equity.

## How to use

```js
const potBeforeRaise = 60;
const raiseSize = 45;
const foldEquity = 0.35;
const potIfCalled = 150;

const need = poker.breakevenRaiseEquity(potBeforeRaise, raiseSize, foldEquity, potIfCalled);
const ev = poker.expectedValueRaise(need, potBeforeRaise, raiseSize, foldEquity, potIfCalled);
console.log(ev); // ≈ 0
```
