# `netPotAfterRake`

Pot minus rake: `pot − rakeFromPot`.

<div className="apiSignature"><code>netPotAfterRake(potChips: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Pot chips **after** rake is removed: `pot − rakeFromPot`. Use in breakdowns next to [`rakeFromPot`](/docs/reference/api/pot-and-ev/rake-from-pot).

## How to use

```js
const pot = 150;
const rakeFraction = 0.05;
const rakeCap = 3;

const rake = poker.rakeFromPot(pot, rakeFraction, rakeCap);
const net = poker.netPotAfterRake(pot, rakeFraction, rakeCap);
console.log({ rake, net }); // net + rake === pot
```
