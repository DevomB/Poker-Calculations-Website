# `netPotAfterRake`

Pot minus rake: `pot − rakeFromPot`.

<div className="apiSignature"><code>netPotAfterRake(potChips: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
```

## When to use

Use when you need pot minus rake: `pot − rakefrompot` in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.netPotAfterRake(/* see signature above */);
console.log(result);
```
