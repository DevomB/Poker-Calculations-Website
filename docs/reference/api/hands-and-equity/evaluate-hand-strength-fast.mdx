# `evaluateHandStrengthFast`

Same encoded strength `number` as [`evaluateHandStrength`](./evaluate-hand-strength), using the in-house **forge** stack evaluator (`poker-calculations-forge`) on Monte Carlo and exact-equity hot paths.

<div className="apiSignature">{`evaluateHandStrengthFast(holeCards: CardInput, board: CardInput): number`}</div>

`CardInput` is `string[]` or `Uint8Array` of deck ids `0..51`. See [Card notation](/docs/concepts/card-notation).

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Use when you need the same sortable strength scalar as `evaluateHandStrength` but want the faster evaluator (for example high-volume sims or batch ranking). Compare paths with [`benchmarkEvaluatorThroughput`](./benchmark-evaluator-throughput) on your hardware.

## How to use

```js
const fast = poker.evaluateHandStrengthFast(['Ah', 'Kh'], ['Qh', 'Jh', 'Th']);
const legacy = poker.evaluateHandStrength(['Ah', 'Kh'], ['Qh', 'Jh', 'Th']);
// fast === legacy for the same spot (same encoding)
```
