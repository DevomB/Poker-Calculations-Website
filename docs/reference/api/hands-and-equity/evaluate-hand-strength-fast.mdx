# `evaluateHandStrengthFast`

Same encoding as `evaluateHandStrength`, using the in-house **forge** stack evaluator used on Monte Carlo and exact hot paths.

<div className="apiSignature">{`evaluateHandStrengthFast(holeCards: string[], board: string[]): string`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Use in performance-sensitive loops (large sim batches, real-time trainers) where you want the same ordering as `evaluateHandStrength` on the fast code path.

## How to use

```js
const fast = poker.evaluateHandStrengthFast(['Ah', 'Kh'], ['Qh', 'Jh', 'Th']);
const legacy = poker.evaluateHandStrength(['Ah', 'Kh'], ['Qh', 'Jh', 'Th']);
// Should match for valid spots; benchmark with benchmarkEvaluatorThroughput
```
