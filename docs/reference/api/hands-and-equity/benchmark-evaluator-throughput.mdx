# `benchmarkEvaluatorThroughput`

Benchmark legacy vs forge evaluator throughput on random 7-card spots.

<div className="apiSignature">{`benchmarkEvaluatorThroughput(iterations?: number): { legacyEvalsPerSecond: number; fastEvalsPerSecond: number; implementation: string }`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Use when tuning server hardware or choosing between `evaluateHandStrength` and `evaluateHandStrengthFast` in production.

## How to use

```js
const bench = poker.benchmarkEvaluatorThroughput(50_000);
console.log(bench.implementation, bench.fastEvalsPerSecond, bench.legacyEvalsPerSecond);
```

`iterations` defaults in native code when omitted.
