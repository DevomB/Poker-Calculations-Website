<div className="apiSignature">{`benchmarkEvaluatorThroughputAsync(iterations?: number): Promise<EvaluatorBenchmarkResult>`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
```

## When to use

Run evaluator benchmarks without blocking the event loop.

## How to use

```js
const poker = require('poker-calculations');

const bench = await poker.benchmarkEvaluatorThroughputAsync(50_000);
```
