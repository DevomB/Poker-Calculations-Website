# `buildPreflopEquityMatrix`

Build a 169×169 preflop equity matrix (row-major `Float64Array`).

<div className="apiSignature">{`buildPreflopEquityMatrix(options?: PreflopMatrixOptions): Float64Array`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Use for heatmaps, correlation studies, or batch preflop analysis.

## How to use

```js
const poker = require('poker-calculations');

const matrix = poker.buildPreflopEquityMatrix({ iterations: 2000, seed: 1, threads: 4 });
console.log(matrix.length); // 169 * 169
```

Each cell uses Monte Carlo with a fixed disjoint-suit matchup for that row/column pair. For `i ≠ j`, **`matrix[j * 169 + i] = 1 - matrix[i * 169 + j]`** (symmetry). See [Numerical semantics](/docs/concepts/numerical-semantics#preflop-equity-matrix).
