# `chubukovSymmetricJamEv`

Symmetric jam toy EV in chips: `equity * (2 * jamStack + dead) - jamStack`.

<div className="apiSignature">{`chubukovSymmetricJamEv(jamStackChips: number, deadMoneyChips: number, equity: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Score **jam-or-fold EV** in the Chubukov symmetric model given jam size, dead money (blinds), and your equity when called. Compare stacks before choosing push/fold in study charts.

## How to use

```js
const jamStack = 4000;
const dead = 150;
const equity = 0.52;

const ev = poker.chubukovSymmetricJamEv(jamStack, dead, equity);
console.log('jam EV:', ev); // positive → jam prints +EV in toy

const maxJam = poker.chubukovMaxSymmetricJamStackChipsBinarySearch(
  equity,
  dead,
  10000
);
console.log('max +EV jam stack:', maxJam);
```
