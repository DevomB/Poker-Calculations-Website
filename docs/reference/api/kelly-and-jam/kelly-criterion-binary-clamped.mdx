# `kellyCriterionBinaryClamped`

Full Kelly clamped to `[0,1]`.

<div className="apiSignature"><code>kellyCriterionBinaryClamped(winProbability: number, netOdds: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Full Kelly clamped to `[0, 1]` so simulators never return negative or >100% bankroll fractions.

## How to use

```js
const f = poker.kellyCriterionBinaryClamped(0.9, 0.5);
console.log(f); // in [0, 1]
```
