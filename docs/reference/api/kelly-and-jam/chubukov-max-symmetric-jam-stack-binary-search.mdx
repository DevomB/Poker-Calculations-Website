# `chubukovMaxSymmetricJamStackBinarySearch`

Largest integer jam stack in `[1, maxStackChips]` with nonnegative symmetric-jam EV using exact HU equity vs a random hand; board 3–5.

<div className="apiSignature">{`chubukovMaxSymmetricJamStackBinarySearch(heroHoleCards: CardInput, boardCards: CardInput, deadMoneyChips: number, maxStackChips: number): number`}</div>

`CardInput` is `string[]` or `Uint8Array` of deck ids `0..51`. See [Card notation](/docs/concepts/card-notation).

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Push/fold study on a **concrete flop/turn/river**: equity comes from `exactHuEquityVsRandomHand`, then binary search finds max +EV jam depth. Board must have 3–5 cards.

## How to use

```js
const hero = ['As', 'Kd'];
const board = ['2h', '7d', '9c'];
const dead = 150;
const maxStack = 5000;

const maxJam = poker.chubukovMaxSymmetricJamStackBinarySearch(
  hero,
  board,
  dead,
  maxStack
);
console.log('max +EV jam:', maxJam);

const eq = poker.exactHuEquityVsRandomHand(hero, board);
console.log('equity vs random:', eq);
```
