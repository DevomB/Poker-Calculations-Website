# `quarterKellyCriterionBinary`

Quarter Kelly fraction.

<div className="apiSignature"><code>quarterKellyCriterionBinary(winProbability: number, netOdds: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Quarter-Kelly stake fraction.

## How to use

```js
const p = 0.55;
const b = 1;
console.log(poker.quarterKellyCriterionBinary(p, b));
```
