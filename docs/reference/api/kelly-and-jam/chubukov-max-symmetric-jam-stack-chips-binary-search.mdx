# `chubukovMaxSymmetricJamStackChipsBinarySearch`

Largest integer jam stack in `[1, maxStackChips]` with nonnegative symmetric-jam EV for the supplied equity.

<div className="apiSignature">{`chubukovMaxSymmetricJamStackChipsBinarySearch(equity: number, deadMoneyChips: number, maxStackChips: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

You already have **jam equity** (from sim or exact HU) and want the deepest integer stack that still has nonnegative toy jam EV. Binary search over `[1, maxStackChips]`.

## How to use

```js
const equity = 0.501;
const dead = 120;
const maxStack = 8000;

const maxProfitableJam = poker.chubukovMaxSymmetricJamStackChipsBinarySearch(
  equity,
  dead,
  maxStack
);
console.log(maxProfitableJam);

console.log(
  poker.chubukovSymmetricJamEv(maxProfitableJam, dead, equity)
);
```
