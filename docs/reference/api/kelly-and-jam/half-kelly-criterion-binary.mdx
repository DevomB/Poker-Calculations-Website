# `halfKellyCriterionBinary`

Half Kelly fraction, floored at 0.

<div className="apiSignature"><code>halfKellyCriterionBinary(winProbability: number, netOdds: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Half-Kelly fraction for binary outcomes — common live-game discipline vs full [`kellyCriterionBinary`](/docs/reference/api/kelly-and-jam/kelly-criterion-binary).

## How to use

```js
const p = 0.55;
const b = 1;

const full = poker.kellyCriterionBinary(p, b);
const half = poker.halfKellyCriterionBinary(p, b);
console.log(half); // max(0, full/2)
```
