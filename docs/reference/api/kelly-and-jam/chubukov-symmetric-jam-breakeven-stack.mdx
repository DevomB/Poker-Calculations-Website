# `chubukovSymmetricJamBreakevenStack`

Toy symmetric jam breakeven stack from dead money and equity (toy HU).

<div className="apiSignature">{`chubukovSymmetricJamBreakevenStack(deadMoneyChips: number, equity: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

In the **symmetric jam toy**, find the stack depth where jamming is neutral given blinds/antes as dead money and your jam equity. `equity > 0.5` returns `Infinity` (jam always +EV in this toy); `equity < 0.5` uses `S = equity·dead/(1−2·equity)`.

## How to use

```js
const dead = 150; // blinds posted
const equity = 0.48;

const breakevenStack = poker.chubukovSymmetricJamBreakevenStack(dead, equity);
console.log('breakeven jam stack:', breakevenStack);

console.log(poker.chubukovSymmetricJamEv(breakevenStack, dead, equity)); // ~0
```
