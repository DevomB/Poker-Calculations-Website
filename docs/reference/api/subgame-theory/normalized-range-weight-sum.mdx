# `normalizedRangeWeightSum`

Sum of range weights.

<div className="apiSignature"><code>normalizedRangeWeightSum(weights: F64VectorInput): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);

```

## When to use

Use when you need sum of range weights in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.normalizedRangeWeightSum(/* see signature above */);
console.log(result);
```
