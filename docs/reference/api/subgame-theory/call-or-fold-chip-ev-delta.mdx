# `callOrFoldChipEvDelta`

Call EV minus fold EV (0).

<div className="apiSignature"><code>callOrFoldChipEvDelta(equity: number, pot: number, toCall: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Chip EV of calling minus folding (fold EV = 0). Equals [`expectedValueCall`](/docs/reference/api/pot-and-ev/expected-value-call).

## How to use

```js
const equity = 0.38;
const pot = 100;
const toCall = 50;

console.log(
  poker.callOrFoldChipEvDelta(equity, pot, toCall),
  poker.expectedValueCall(equity, pot, toCall),
);
```
