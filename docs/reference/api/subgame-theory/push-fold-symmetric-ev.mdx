# `pushFoldSymmetricEv`

Symmetric push/fold chip EV.

<div className="apiSignature"><code>pushFoldSymmetricEv(equity: number, jamStackChips: number, deadMoneyChips: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Symmetric push/fold chip EV — alias of [`chubukovSymmetricJamEv`](/docs/reference/api/kelly-and-jam/chubukov-symmetric-jam-ev).

## How to use

```js
const equity = 0.48;
const jamStack = 100;
const deadMoney = 15;

console.log(
  poker.pushFoldSymmetricEv(equity, jamStack, deadMoney),
  poker.chubukovSymmetricJamEv(equity, jamStack, deadMoney),
);
```
