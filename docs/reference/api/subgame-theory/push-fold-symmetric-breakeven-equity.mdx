# `pushFoldSymmetricBreakevenEquity`

Breakeven equity for symmetric jam.

<div className="apiSignature"><code>pushFoldSymmetricBreakevenEquity(jamStackChips: number, deadMoneyChips: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Equity where symmetric jam EV is zero.

## How to use

```js
const jamStack = 100;
const deadMoney = 15;

const eq = poker.pushFoldSymmetricBreakevenEquity(jamStack, deadMoney);
console.log(poker.pushFoldSymmetricEv(eq, jamStack, deadMoney)); // ≈ 0
```
