# `openRaiseBreakevenFoldEquity`

Fold equity needed to profit open-raising air.

<div className="apiSignature"><code>openRaiseBreakevenFoldEquity(potBeforeHeroBet: number, heroOpenRaiseSize: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Fold equity needed to profit open-raising pure air — same as [`breakevenFoldEquityPureBluff`](/docs/reference/api/fold-equity/breakeven-fold-equity-pure-bluff).

## How to use

```js
const pot = 15; // blinds
const openSize = 25;

console.log(
  poker.openRaiseBreakevenFoldEquity(pot, openSize),
  poker.breakevenFoldEquityPureBluff(pot, openSize),
);
```
