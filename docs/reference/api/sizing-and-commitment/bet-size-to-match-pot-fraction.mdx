# `betSizeToMatchPotFraction`

Bet size for target pot fraction (inverse of `betAsPotFraction`).

<div className="apiSignature"><code>betSizeToMatchPotFraction(potBeforeBet: number, targetFraction: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Inverse of [`betAsPotFraction`](/docs/reference/api/sizing-and-commitment/bet-as-pot-fraction): bet size that achieves target pot fraction.

## How to use

```js
const potBeforeBet = 100;
const targetFraction = 0.5;

const bet = poker.betSizeToMatchPotFraction(potBeforeBet, targetFraction);
console.log(poker.betAsPotFraction(potBeforeBet, bet)); // 0.5
```
