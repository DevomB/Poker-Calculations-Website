# `commitmentRatioAfterBet`

Fraction of stack committed by bet.

<div className="apiSignature"><code>commitmentRatioAfterBet(betSize: number, effectiveStackBeforeBet: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Fraction of effective stack put in with the bet: `bet / stack`.

## How to use

```js
const betSize = 80;
const stack = 320;

console.log(poker.commitmentRatioAfterBet(betSize, stack)); // 0.25
```
