# `sprAfterRaise`

SPR after open/raise heads-up.

<div className="apiSignature"><code>sprAfterRaise(potBeforeRaise: number, raiseSize: number, effectiveStackBeforeRaise: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

SPR after an open/raise that is called — same geometry as [`sprAfterBet`](/docs/reference/api/sizing-and-commitment/spr-after-bet).

## How to use

```js
const sprBet = poker.sprAfterBet(100, 50, 400);
const sprRaise = poker.sprAfterRaise(100, 50, 400);
console.log(sprBet === sprRaise); // true
```
