# `sprAfterBet`

SPR after hero bets heads-up.

<div className="apiSignature"><code>sprAfterBet(potBeforeBet: number, betSize: number, effectiveStackBeforeBet: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Stack-to-pot ratio **after** hero bets and is called heads-up: `(stack − bet) / (pot + 2×bet)`.

## How to use

```js
const potBeforeBet = 100;
const betSize = 50;
const stack = 400;

const spr = poker.sprAfterBet(potBeforeBet, betSize, stack);
console.log(spr); // 350 / 200 = 1.75
```
