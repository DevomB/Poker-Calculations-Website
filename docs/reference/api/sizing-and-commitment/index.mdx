# Sizing & commitment

Bet sizing as a **fraction of pot**, **SPR after a call**, and how much of the effective stack a call consumes. Assumes heads-up single-call geometry for `sprAfterCall`.

Pair with [Pot & EV](/docs/reference/api/pot-and-ev/) for breakeven equity and with [Stacks & display](/docs/reference/api/stacks-and-display/) for pre-call SPR.

| Export | Summary |
| --- | --- |
| [`betAsPotFraction`](/docs/reference/api/sizing-and-commitment/bet-as-pot-fraction) | Bet as fraction of pot. |
| [`sprAfterCall`](/docs/reference/api/sizing-and-commitment/spr-after-call) | SPR after HU single call. |
| [`commitmentRatio`](/docs/reference/api/sizing-and-commitment/commitment-ratio) | Fraction of stack put in to call. |
| [`sprAfterBet`](/docs/reference/api/sizing-and-commitment/spr-after-bet) | SPR after hero bets heads-up. |
| [`sprAfterRaise`](/docs/reference/api/sizing-and-commitment/spr-after-raise) | SPR after open/raise heads-up. |
| [`commitmentRatioAfterBet`](/docs/reference/api/sizing-and-commitment/commitment-ratio-after-bet) | Fraction of stack committed by bet. |
| [`betSizeToMatchPotFraction`](/docs/reference/api/sizing-and-commitment/bet-size-to-match-pot-fraction) | Bet size for target pot fraction (inverse of `betAsPotFraction`). |
