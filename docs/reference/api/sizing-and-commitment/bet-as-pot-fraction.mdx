# `betAsPotFraction`

Bet as fraction of pot.

<div className="apiSignature">{`betAsPotFraction(potBeforeBet: number, betSize: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Label bets as **pot fractions** (half-pot, 75%, pot, overbet) in replays and solvers. Input is pot before the bet and the bet size in chips.

## How to use

```js
const pot = 120;
const bet = 90;
console.log(poker.betAsPotFraction(pot, bet)); // 0.75 — 75% pot bet

console.log(poker.bluffToValueRatio(pot, bet));
console.log(poker.alphaFrequency(pot, bet));
```
