# `fullHouseMadeFlopToRiverExactProbability`

Exact full house+ by river.

Card lists use `CardInput` (`string[]` or packed `Uint8Array`). See [Card notation](/docs/concepts/card-notation).

<div className="apiSignature"><code>fullHouseMadeFlopToRiverExactProbability(heroHoleCards: CardInput, flopThree: CardInput, knownDead: CardInput): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
```

## When to use

Use when you need exact full house+ by river in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.fullHouseMadeFlopToRiverExactProbability(/* see signature above */);
```
