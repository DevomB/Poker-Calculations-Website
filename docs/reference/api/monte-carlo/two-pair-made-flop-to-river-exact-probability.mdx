# `twoPairMadeFlopToRiverExactProbability`

Exact two pair+ by river.

Card lists use `CardInput` (`string[]` or packed `Uint8Array`). See [Card notation](/docs/concepts/card-notation).

<div className="apiSignature"><code>twoPairMadeFlopToRiverExactProbability(heroHoleCards: CardInput, flopThree: CardInput, knownDead: CardInput): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Exact **two pair or better** by the river.

## How to use

```js
const hole = ['Ah', 'Kd'];
const flop = ['Ac', '7d', '9c'];

console.log(poker.twoPairMadeFlopToRiverExactProbability(hole, flop, []));
```
