# `sidePotLayersTotalChips`

Sum of `potChips` across side-pot layers.

<div className="apiSignature">{`sidePotLayersTotalChips(layers: SidePotLayer[]): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Quick **total chips in play** across main and side pots after building a ladder—sanity checks, pot display, and comparing to table `pot` field before EV attribution.

## How to use

```js
const layers = poker.sidePotLadderFromCommitments([200, 500, 500, 100]);
const total = poker.sidePotLayersTotalChips(layers);
console.log('total in all layers:', total);

layers.forEach((layer, i) => {
  console.log(`layer ${i}:`, layer.potChips);
});
```
