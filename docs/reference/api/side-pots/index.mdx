# Side pots

Build **side-pot ladders** from per-player commitments, sum layer sizes, and compute **chip EV** per player when each layer has its own win-equity matrix.

Layer equities: each column (layer) must sum to `1` across players.

| Export | Summary |
| --- | --- |
| [`sidePotLadderFromCommitments`](/docs/reference/api/side-pots/side-pot-ladder-from-commitments) | Main + side layers from commitments. |
| [`layeredPotChipEvFromEquities`](/docs/reference/api/side-pots/layered-pot-chip-ev-from-equities) | Chip EV per player from layer equities. |
| [`sidePotLayersTotalChips`](/docs/reference/api/side-pots/side-pot-layers-total-chips) | Sum of `potChips` across layers. |
| [`sidePotLayerCount`](/docs/reference/api/side-pots/side-pot-layer-count) | Number of side-pot layers. |
| [`sidePotBreakevenCallEquity`](/docs/reference/api/side-pots/side-pot-breakeven-call-equity) | Breakeven equity for hero side-pot layer. |
| [`layeredPotChipEvFromEquitiesWithRake`](/docs/reference/api/side-pots/layered-pot-chip-ev-from-equities-with-rake) | Layered chip EV with rake per pot. |
