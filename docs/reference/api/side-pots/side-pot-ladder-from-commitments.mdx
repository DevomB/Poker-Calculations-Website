# `sidePotLadderFromCommitments`

Build the **main pot and side pots** from per-player committed chips. Each layer is a `SidePotLayer`: `{ potChips, playerCapContribution[] }`.

<div className="apiSignature"><code>sidePotLadderFromCommitments(committedChips: number[]): SidePotLayer[]</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

**All-in multiway** pots with unequal stacks—split committed chips into contestable layers before assigning per-layer equities. Feed layer sizes into [`layeredPotChipEvFromEquities`](/docs/reference/api/side-pots/layered-pot-chip-ev-from-equities); verify totals with [`sidePotLayersTotalChips`](/docs/reference/api/side-pots/side-pot-layers-total-chips).

## How to use

```js
const poker = require('poker-calculations');

const committedChips = [100, 250, 250, 80];
const layers = poker.sidePotLadderFromCommitments(committedChips);

layers.forEach((layer, i) => {
  console.log(`layer ${i}`, layer.potChips, layer.playerCapContribution);
});

const total = poker.sidePotLayersTotalChips(layers);
console.log({ layers, totalCommitted: committedChips.reduce((a, b) => a + b, 0), totalInLayers: total });
```

**Returns** `SidePotLayer[]` — ordered ladder; `playerCapContribution[j]` is player `j`’s cap into that layer.
