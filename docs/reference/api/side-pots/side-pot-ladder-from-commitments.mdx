# `sidePotLadderFromCommitments`

Main + side layers from per-player committed chips.

<div className="apiSignature">{`sidePotLadderFromCommitments(committedChips: number[]): SidePotLayer[]`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

All-in side pots: pass each player’s **total chips committed** this hand to get ordered layers `{ potChips, playerCapContribution[] }`. Feed layers into equity-by-layer EV or display pot breakdown UIs.

## How to use

```js
const committed = [100, 300, 300, 50]; // seat 0 short, seat 3 shorter
const layers = poker.sidePotLadderFromCommitments(committed);
console.log(layers);

console.log(poker.sidePotLayersTotalChips(layers));
```
