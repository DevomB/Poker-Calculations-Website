# `sidePotBreakevenCallEquity`

Breakeven equity for hero side-pot layer.

<div className="apiSignature"><code>sidePotBreakevenCallEquity(layerPotChips: number, toCall: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Breakeven equity for a call into a **single layer** pot: `toCall / (layerPot + toCall)`.

## How to use

```js
const layerPot = 150;
const toCall = 50;

console.log(poker.sidePotBreakevenCallEquity(layerPot, toCall)); // 50/200
```
