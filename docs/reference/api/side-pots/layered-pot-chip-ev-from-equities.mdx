# `layeredPotChipEvFromEquities`

Per-player **chip EV** across side-pot layers from layer pot sizes and a per-player, per-layer equity matrix (each layer column sums to `1`).

<div className="apiSignature"><code>layeredPotChipEvFromEquities(layerPotChips: number[], equityPlayerByLayer: number[][]): number[]</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

After [`sidePotLadderFromCommitments`](/docs/reference/api/side-pots/side-pot-ladder-from-commitments), you have layer pot sizes and **equity per player in each layer** (only eligible players have mass; columns sum to `1`). Use Monte Carlo or exact equity per layer, then aggregate chip EV per seat.

## How to use

```js
const poker = require('poker-calculations');

const committedChips = [100, 250, 250];
const layers = poker.sidePotLadderFromCommitments(committedChips);
const layerPotChips = layers.map((l) => l.potChips);

const equityPlayerByLayer = [
  [0.55, 0.40],
  [0.30, 0.35],
  [0.15, 0.25],
];

const ev = poker.layeredPotChipEvFromEquities(layerPotChips, equityPlayerByLayer);
console.log(ev);

const totalPot = poker.sidePotLayersTotalChips(layers);
const evSum = ev.reduce((a, b) => a + b, 0);
console.log({ ev, totalPot, evSum });
```

**Returns** `number[]` — chip EV per player, length = number of rows in `equityPlayerByLayer`. Throws if column sums deviate from `1` or dimensions mismatch.
