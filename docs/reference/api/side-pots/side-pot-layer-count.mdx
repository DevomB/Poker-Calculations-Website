# `sidePotLayerCount`

Number of side-pot layers.

<div className="apiSignature"><code>sidePotLayerCount(committedChips: F64VectorInput): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Number of side-pot layers from committed chip counts — same as `sidePotLadderFromCommitments(...).length`.

## How to use

```js
const committed = [100, 200, 500];
const layers = poker.sidePotLadderFromCommitments(committed);
console.log(poker.sidePotLayerCount(committed), layers.length);
```
