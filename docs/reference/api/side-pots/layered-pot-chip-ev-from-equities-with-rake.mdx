# `layeredPotChipEvFromEquitiesWithRake`

Layered chip EV with rake per pot.

<div className="apiSignature"><code>layeredPotChipEvFromEquitiesWithRake(layerPotChips: F64VectorInput, equityPlayerByLayer: number[][] | Float64Array, rakeFraction: number, rakeCap: number, returnFormat?: F64ReturnFormat): number[] | Float64Array</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Like [`layeredPotChipEvFromEquities`](/docs/reference/api/side-pots/layered-pot-chip-ev-from-equities) but rake is applied per layer won.

## How to use

```js
const layerPots = [150, 400];
const equities = [
  [0.55, 0.10],
  [0.30, 0.60],
  [0.15, 0.30],
];

const ev = poker.layeredPotChipEvFromEquitiesWithRake(
  layerPots,
  equities,
  0.05,
  3,
);
const ev0 = poker.layeredPotChipEvFromEquities(layerPots, equities);
console.log({ ev, ev0 });
```
