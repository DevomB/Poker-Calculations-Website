# `icmPayoutStructureGini`

Inequality index of payout ladder.

<div className="apiSignature"><code>icmPayoutStructureGini(payouts: F64VectorInput): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Inequality index of the payout ladder (0 when all payouts equal).

## How to use

```js
const equal = poker.icmPayoutStructureGini([100, 100, 100]);
const steep = poker.icmPayoutStructureGini([10000, 3000, 1000]);
console.log({ equal, steep });
```
