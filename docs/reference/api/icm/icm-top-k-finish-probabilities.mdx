# `icmTopKFinishProbabilities`

Per-player probability of finishing in one of the first `k` places; `k` in `1..stacks.length`.

<div className="apiSignature">{`icmTopKFinishProbabilities(stacks: number[], k: number): number[]`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Prize pools that pay **top k** (e.g. final-table chop thinking, satellite “top 3 win seat”) need summed placement mass without manual matrix reduction.

## How to use

```js
const stacks = [8000, 7000, 5000, 2000];
const pTop2 = poker.icmTopKFinishProbabilities(stacks, 2);
console.log(pTop2); // P(finish 1st or 2nd) per player

const pTop3 = poker.icmTopKFinishProbabilities(stacks, 3);
console.log(pTop3);
```
