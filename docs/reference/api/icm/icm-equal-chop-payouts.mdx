# `icmEqualChopPayouts`

Equal split of prize pool per seat.

<div className="apiSignature"><code>icmEqualChopPayouts(payouts: F64VectorInput, returnFormat?: F64ReturnFormat): number[] | Float64Array</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Split the prize pool **equally** across seats (negotiated equal chop).

## How to use

```js
const payouts = [5000, 3000, 1500, 500];
const equal = poker.icmEqualChopPayouts(payouts);
console.log(equal); // each 2500; sum === 10000
```
