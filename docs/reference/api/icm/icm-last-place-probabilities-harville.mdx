# `icmLastPlaceProbabilitiesHarville`

Harville probability each player finishes last (placement matrix last column).

<div className="apiSignature">{`icmLastPlaceProbabilitiesHarville(stacks: number[]): number[]`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Short stacks care about **bust-out risk** before win equity—last-place probabilities from Harville highlight who is most likely to exit next. Equivalent to the last column of `icmHarvillePlacementProbabilities`.

## How to use

```js
const stacks = [15000, 12000, 4000, 2000];
const pLast = poker.icmLastPlaceProbabilitiesHarville(stacks);
console.log(pLast);

const matrix = poker.icmHarvillePlacementProbabilities(stacks);
console.log('last column from matrix:', matrix.map((row) => row[stacks.length - 1]));
```
