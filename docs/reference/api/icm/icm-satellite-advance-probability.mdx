# `icmSatelliteAdvanceProbability`

Per-seat probability of cashing satellite.

<div className="apiSignature"><code>icmSatelliteAdvanceProbability(stacks: F64VectorInput, paidPlaces: number, returnFormat?: F64ReturnFormat): number[] | Float64Array</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);

```

## When to use

Use when you need per-seat probability of cashing satellite in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.icmSatelliteAdvanceProbability(/* see signature above */);
console.log(result);
```
