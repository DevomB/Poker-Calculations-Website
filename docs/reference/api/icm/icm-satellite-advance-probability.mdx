# `icmSatelliteAdvanceProbability`

Per-seat probability of cashing satellite.

<div className="apiSignature"><code>icmSatelliteAdvanceProbability(stacks: F64VectorInput, paidPlaces: number, returnFormat?: F64ReturnFormat): number[] | Float64Array</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Per-seat probability of finishing in the money in a **satellite** (`paidPlaces` paid spots).

## How to use

```js
const stacks = [8000, 6000, 4000, 2000];
const paidPlaces = 2;

const advance = poker.icmSatelliteAdvanceProbability(stacks, paidPlaces);
console.log(advance); // each in [0, 1]
```
