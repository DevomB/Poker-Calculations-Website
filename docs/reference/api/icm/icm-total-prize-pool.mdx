# `icmTotalPrizePool`

Sum of payout ladder.

<div className="apiSignature"><code>icmTotalPrizePool(payouts: F64VectorInput): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Sum of the payout ladder.

## How to use

```js
const payouts = [100, 50, 30];
console.log(poker.icmTotalPrizePool(payouts)); // 180
```
