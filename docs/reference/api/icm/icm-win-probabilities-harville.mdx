# `icmWinProbabilitiesHarville`

Harville first-place probabilities.

<div className="apiSignature">{`icmWinProbabilitiesHarville(stacks: F64VectorInput, returnFormat?: F64ReturnFormat): number[] | Float64Array`}</div>

Accepts `number[]` or `Float64Array`. Optional `'float64'` return format. See [Float64 ICM input](/docs/concepts/float64-icm-input).

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Tournament tools need **P(win)** under Harville from current stacks—bubble decisions, deal calculators, and first-stage ICM before full placement matrix work.

## How to use

```js
const stacks = [12000, 9000, 6000, 3000];
const pWin = poker.icmWinProbabilitiesHarville(stacks);
console.log(pWin); // sums to 1

const payouts = [10000, 6000, 4000, 2000];
console.log(poker.icmExpectedPayouts(stacks, payouts));
```
