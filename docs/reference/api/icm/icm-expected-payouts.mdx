# `icmExpectedPayouts`

**ICM expected prize** per player from Harville placement probabilities and a prize vector (length = players).

<div className="apiSignature"><code>icmExpectedPayouts(stacks: number[], payouts: number[]): number[]</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Convert stacks and a **payout ladder** into dollar (or chip-prize) equity per seat—core MTT ICM output. `payouts[i]` is the prize for finish `i+1` (first place at index `0`). Building block for [`icmPairwiseBubbleFactor`](/docs/reference/api/icm/icm-pairwise-bubble-factor).

## How to use

```js
const poker = require('poker-calculations');

const stacks = [12000, 8000, 5000];
const payouts = [400, 250, 150];

const icm = poker.icmExpectedPayouts(stacks, payouts);
console.log(icm);

const prizePool = payouts.reduce((a, b) => a + b, 0);
const icmSum = icm.reduce((a, b) => a + b, 0);
console.log({ icm, prizePool, icmSum });
```

**Returns** `number[]` — expected payout per player; sums to total prizes when `payouts` covers all places.
