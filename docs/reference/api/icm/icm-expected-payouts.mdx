# `icmExpectedPayouts`

ICM expected payouts (prize vector length = players).

<div className="apiSignature">{`icmExpectedPayouts(stacks: F64VectorInput, payouts: F64VectorInput, returnFormat?: F64ReturnFormat): number[] | Float64Array`}</div>

See [Float64 ICM input](/docs/concepts/float64-icm-input).

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Convert stacks and a **fixed payout ladder** into expected dollar (or chip) equity per seat—deal making, bubble pressure, and `icmPairwiseBubbleFactor` all build on this vector.

## How to use

```js
const stacks = [10000, 7500, 5000, 2500];
const payouts = [5000, 3000, 1500, 500];

const ev = poker.icmExpectedPayouts(stacks, payouts);
console.log(ev); // expected prize per player; sum ≈ sum(payouts)

console.log(poker.icmWinProbabilitiesHarville(stacks));
```
