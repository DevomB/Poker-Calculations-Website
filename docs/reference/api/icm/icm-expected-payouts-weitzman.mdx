# `icmExpectedPayoutsWeitzman`

Expected ICM payouts using independent Weitzman chip utility (`stack^alpha`).

<div className="apiSignature">{`icmExpectedPayoutsWeitzman(stacks, payouts, alpha?, returnFormat?): number[] | Float64Array`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

**Not Harville placement.** Splits each prize tier independently using chip utility `stack^alpha` (default `alpha = 2`). Use when you want a Weitzman-style model instead of [`icmExpectedPayouts`](/docs/reference/api/icm/icm-expected-payouts) Harville math.

## How to use

```js
const poker = require('poker-calculations');

const ev = poker.icmExpectedPayoutsWeitzman([1000, 800, 600], [500, 300, 200], 2);
```
