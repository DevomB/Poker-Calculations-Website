# `icmHarvillePlacementProbabilities`

Full Harville **placement matrix** `[player][finishRank]`; rank `0` is first place.

<div className="apiSignature"><code>icmHarvillePlacementProbabilities(stacks: number[]): number[][]</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Full **finish-order distribution** for MTT ICM study—each row is one player, each column a finish rank. Feeds [`icmTopKFinishProbabilities`](/docs/reference/api/icm/icm-top-k-finish-probabilities) and [`icmLastPlaceProbabilitiesHarville`](/docs/reference/api/icm/icm-last-place-probabilities-harville).

## How to use

```js
const poker = require('poker-calculations');

const stacks = [10000, 6000, 4000];
const matrix = poker.icmHarvillePlacementProbabilities(stacks);

console.log(matrix[0][0]);
console.log(matrix[0].reduce((a, b) => a + b, 0));

const winOnly = poker.icmWinProbabilitiesHarville(stacks);
console.log({ firstPlaceCol: matrix.map((row) => row[0]), winOnly });
```

**Returns** `number[][]` — `n × n` for `n` players; each row sums to `1`.
