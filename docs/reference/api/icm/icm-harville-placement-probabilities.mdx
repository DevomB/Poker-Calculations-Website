# `icmHarvillePlacementProbabilities`

Full Harville placement matrix `[player][finishRank]` (rank 0 = first).

<div className="apiSignature">{`icmHarvillePlacementProbabilities(stacks: number[]): number[][]`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

You need the full **finish distribution** per player (1st through nth) for custom payout structures or visualization. Row `i` is player `i`; column `0` is first place.

## How to use

```js
const stacks = [5000, 5000, 2500];
const matrix = poker.icmHarvillePlacementProbabilities(stacks);

console.log('P(player 0 wins):', matrix[0][0]);
console.log('P(player 2 last):', matrix[2][2]);

console.log(poker.icmTopKFinishProbabilities(stacks, 2));
```
