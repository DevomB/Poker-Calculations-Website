# ICM

**Harville** placement and win probabilities, expected payouts from a prize vector, top-k and last-place conveniences, and pairwise **bubble factors** from payout finite differences.

Stack inputs are chip counts; payouts length must match player count. Not the same as `normalizedStackFractions` (raw chip share only).

Stacks (and some outputs) accept `Float64Array` and optional `returnFormat: 'float64'`. See [Float64 ICM input](/docs/concepts/float64-icm-input).

| Export | Summary |
| --- | --- |
| [`icmWinProbabilitiesHarville`](/docs/reference/api/icm/icm-win-probabilities-harville) | First-place probabilities. |
| [`icmHarvillePlacementProbabilities`](/docs/reference/api/icm/icm-harville-placement-probabilities) | Full `n×n` placement matrix. |
| [`icmTopKFinishProbabilities`](/docs/reference/api/icm/icm-top-k-finish-probabilities) | P(finish in top `k`). |
| [`icmLastPlaceProbabilitiesHarville`](/docs/reference/api/icm/icm-last-place-probabilities-harville) | P(finish last). |
| [`icmExpectedPayouts`](/docs/reference/api/icm/icm-expected-payouts) | Expected $ per seat. |
| [`icmPairwiseBubbleFactor`](/docs/reference/api/icm/icm-pairwise-bubble-factor) | Loss/gain ratio on payouts. |
