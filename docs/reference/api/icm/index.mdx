# ICM

**Harville** placement and win probabilities, expected payouts from a prize vector, top-k and last-place conveniences, and pairwise **bubble factors** from payout finite differences.

Stack inputs are chip counts; payouts length must match player count. Not the same as `normalizedStackFractions` (raw chip share only).

| Export | Summary |
| --- | --- |
| [`icmWinProbabilitiesHarville`](./icm-win-probabilities-harville) | First-place probabilities. |
| [`icmHarvillePlacementProbabilities`](./icm-harville-placement-probabilities) | Full `n×n` placement matrix. |
| [`icmTopKFinishProbabilities`](./icm-top-k-finish-probabilities) | P(finish in top `k`). |
| [`icmLastPlaceProbabilitiesHarville`](./icm-last-place-probabilities-harville) | P(finish last). |
| [`icmExpectedPayouts`](./icm-expected-payouts) | Expected $ per seat. |
| [`icmPairwiseBubbleFactor`](./icm-pairwise-bubble-factor) | Loss/gain ratio on payouts. |
