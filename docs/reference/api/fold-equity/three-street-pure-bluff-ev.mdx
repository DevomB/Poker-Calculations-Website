# `threeStreetPureBluffEv`

Three-street pure-bluff chip EV.

<div className="apiSignature"><code>threeStreetPureBluffEv(potBeforeStreet1: number, betStreet1: number, betStreet2: number, betStreet3: number, foldEquityStreet1: number, foldEquityStreet2: number, foldEquityStreet3: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Three-street pure bluff with independent fold equities per street.

## How to use

```js
const ev = poker.threeStreetPureBluffEv(
  50, 35, 90, 180,
  0.4, 0.35, 0.3,
);
console.log('three-street bluff EV:', ev);
```
