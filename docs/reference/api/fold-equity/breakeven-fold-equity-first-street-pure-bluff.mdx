# `breakevenFoldEquityFirstStreetPureBluff`

Breakeven first-street FE given second-street FE (may lie outside [0,1]).

<div className="apiSignature">{`breakevenFoldEquityFirstStreetPureBluff(potBeforeStreet1: number, betStreet1: number, betStreet2: number, foldEquityStreet2: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Plan a two-street bluff **backward from turn fold assumptions**: given expected fold equity on the turn, what flop fold rate makes the line neutral? Useful when turn card population tendencies are better known than flop.

## How to use

```js
const fe1 = poker.breakevenFoldEquityFirstStreetPureBluff(
  55,
  35,
  95,
  0.4
);
console.log('breakeven flop FE:', fe1);

console.log(poker.twoStreetPureBluffEv(55, 35, 95, fe1, 0.4));
```
