# `breakevenFoldEquitySecondStreetPureBluff`

Breakeven second-street FE given first-street FE (may lie outside [0,1]).

<div className="apiSignature">{`breakevenFoldEquitySecondStreetPureBluff(potBeforeStreet1: number, betStreet1: number, betStreet2: number, foldEquityStreet1: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

You committed to a flop bluff with estimated **first-street fold equity** and want the breakeven fold rate on the turn barrel. Result may fall outside `[0, 1]` if the flop bet already makes the plan −EV.

## How to use

```js
const fe2 = poker.breakevenFoldEquitySecondStreetPureBluff(
  70,
  45,
  100,
  0.5
);
console.log('need turn fold equity:', fe2);

console.log(poker.twoStreetPureBluffEv(70, 45, 100, 0.5, fe2)); // ~0 at breakeven
```
