# `twoStreetPureBluffEv`

Two-street pure-bluff chip EV with independent `fe1`, `fe2`.

<div className="apiSignature">{`twoStreetPureBluffEv(potBeforeStreet1: number, betStreet1: number, betStreet2: number, foldEquityStreet1: number, foldEquityStreet2: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Evaluate a **double-barrel air line** when you assign separate fold probabilities on street 1 and street 2 (e.g. from stats). Positive EV means the two-street bluff plan beats folding out both barrels.

## How to use

```js
const ev = poker.twoStreetPureBluffEv(
  60,  // pot before flop bet
  40,  // flop bet
  110, // turn bet
  0.45,
  0.38
);
console.log('two-street bluff EV:', ev);

const fe2 = poker.breakevenFoldEquitySecondStreetPureBluff(60, 40, 110, 0.45);
console.log('breakeven turn FE if flop FE=45%:', fe2);
```
