# `twoStreetPureBluffEv`

Chip **EV** of a two-street pure bluff with independent fold equities `foldEquityStreet1` and `foldEquityStreet2` (air if called through).

<div className="apiSignature"><code>twoStreetPureBluffEv(potBeforeStreet1: number, betStreet1: number, betStreet2: number, foldEquityStreet1: number, foldEquityStreet2: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Evaluate a **double-barrel bluff** when you assign separate fold rates per street. Positive EV means the sequence beats not betting; compare breakeven FE via [`twoStreetPureBluffSameFoldEquity`](/docs/reference/api/fold-equity/two-street-pure-bluff-same-fold-equity) or street-specific breakeven helpers.

## How to use

```js
const poker = require('poker-calculations');

const potBeforeStreet1 = 50;
const betStreet1 = 30;
const betStreet2 = 70;
const fe1 = 0.45;
const fe2 = 0.35;

const ev = poker.twoStreetPureBluffEv(
  potBeforeStreet1,
  betStreet1,
  betStreet2,
  fe1,
  fe2,
);
console.log(ev);

const breakevenFe2 = poker.breakevenFoldEquitySecondStreetPureBluff(
  potBeforeStreet1,
  betStreet1,
  betStreet2,
  fe1,
);
console.log({ ev, breakevenFe2GivenFe1: breakevenFe2 });
```

**Returns** `number` — chip EV of the two-bet sequence (can be negative).
