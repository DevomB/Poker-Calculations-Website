# `twoStreetPureBluffSameFoldEquity`

Same FE both streets, pure air; may return `NaN` if no root in [0,1].

<div className="apiSignature">{`twoStreetPureBluffSameFoldEquity(potBeforeStreet1: number, betStreet1: number, betStreet2: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Two-barrel pure bluff plan where **fold frequency is the same** on flop and turn. Solves for breakeven FE; returns `NaN` when no solution exists in `[0, 1]` (line may be inherently −EV).

## How to use

```js
const pot0 = 50;
const bet1 = 35;
const bet2 = 90;

const fe = poker.twoStreetPureBluffSameFoldEquity(pot0, bet1, bet2);
console.log('breakeven FE each street:', fe);

if (!Number.isNaN(fe)) {
  console.log(poker.twoStreetPureBluffEv(pot0, bet1, bet2, fe, fe));
}
```
