# `breakevenFoldEquitySemiBluffWithAnte`

Semi-bluff fold equity with ante.

<div className="apiSignature"><code>breakevenFoldEquitySemiBluffWithAnte(potBeforeHeroBet: number, heroBetSize: number, equityWhenCalled: number, totalPotIfCalled: number, anteToPost: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Semi-bluff breakeven fold equity with ante in the won-pot model.

## How to use

```js
const fe = poker.breakevenFoldEquitySemiBluffWithAnte(
  100, 50,
  0.35,  // equity when called
  200,   // total pot if called
  5,     // ante
);
console.log(fe);
```
