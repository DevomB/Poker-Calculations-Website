# `threeStreetPureBluffSameFoldEquity`

Same fold equity three streets, pure air.

<div className="apiSignature"><code>threeStreetPureBluffSameFoldEquity(potBeforeStreet1: number, betStreet1: number, betStreet2: number, betStreet3: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Three-barrel air with the **same** fold equity each street (extension of two-street same-FE model).

## How to use

```js
const ev = poker.threeStreetPureBluffSameFoldEquity(
  50,  // pot before flop bet
  35,  // flop bet
  90,  // turn bet
  180, // river bet
);
console.log(ev);
```
