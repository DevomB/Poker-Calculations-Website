# `breakevenFoldEquityPureBluff`

FE when equity if called is 0.

<div className="apiSignature">{`breakevenFoldEquityPureBluff(potBeforeHeroBet: number, heroBetOrCallSize: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Pure air: how often villain must **fold** for your bet to break even if you always lose when called. Compare to population fold-to-bet stats or `betaBinomialFoldPosterior` beliefs.

## How to use

```js
const pot = 100;
const bet = 66;

const fe = poker.breakevenFoldEquityPureBluff(pot, bet);
console.log(fe); // ≈ bet / (pot + bet)

const alpha = poker.alphaFrequency(pot, bet);
console.log('alpha matches pure-bluff BE:', alpha);
```
