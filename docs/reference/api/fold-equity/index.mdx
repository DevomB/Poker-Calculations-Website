# Fold equity

Breakeven **fold equity** for pure bluffs and semi-bluffs (with optional rake), plus two-street pure-bluff models with independent or tied fold frequencies.

Use when sizing bluffs or semi-bluffs and comparing required villain fold % to population or player-model estimates.

| Export | Summary |
| --- | --- |
| [`breakevenFoldEquityPureBluff`](./breakeven-fold-equity-pure-bluff) | FE when equity if called is 0. |
| [`breakevenFoldEquitySemiBluff`](./breakeven-fold-equity-semi-bluff) | Two-outcome semi-bluff model. |
| [`breakevenFoldEquitySemiBluffWithRake`](./breakeven-fold-equity-semi-bluff-with-rake) | Semi-bluff FE with rake on called pot. |
| [`breakevenFoldEquityPureBluffWithRake`](./breakeven-fold-equity-pure-bluff-with-rake) | Pure-bluff FE with rake. |
| [`twoStreetPureBluffSameFoldEquity`](./two-street-pure-bluff-same-fold-equity) | Same FE both streets; may be `NaN`. |
| [`twoStreetPureBluffEv`](./two-street-pure-bluff-ev) | Two-street EV with independent FE. |
| [`breakevenFoldEquitySecondStreetPureBluff`](./breakeven-fold-equity-second-street-pure-bluff) | Breakeven 2nd-street FE given 1st. |
| [`breakevenFoldEquityFirstStreetPureBluff`](./breakeven-fold-equity-first-street-pure-bluff) | Breakeven 1st-street FE given 2nd. |
