# Fold equity

Breakeven **fold equity** for pure bluffs and semi-bluffs (with optional rake), plus two-street pure-bluff models with independent or tied fold frequencies.

Use when sizing bluffs or semi-bluffs and comparing required villain fold % to population or player-model estimates.

| Export | Summary |
| --- | --- |
| [`breakevenFoldEquityPureBluff`](/docs/reference/api/fold-equity/breakeven-fold-equity-pure-bluff) | FE when equity if called is 0. |
| [`breakevenFoldEquitySemiBluff`](/docs/reference/api/fold-equity/breakeven-fold-equity-semi-bluff) | Two-outcome semi-bluff model. |
| [`breakevenFoldEquitySemiBluffWithRake`](/docs/reference/api/fold-equity/breakeven-fold-equity-semi-bluff-with-rake) | Semi-bluff FE with rake on called pot. |
| [`breakevenFoldEquityPureBluffWithRake`](/docs/reference/api/fold-equity/breakeven-fold-equity-pure-bluff-with-rake) | Pure-bluff FE with rake. |
| [`twoStreetPureBluffSameFoldEquity`](/docs/reference/api/fold-equity/two-street-pure-bluff-same-fold-equity) | Same FE both streets; may be `NaN`. |
| [`twoStreetPureBluffEv`](/docs/reference/api/fold-equity/two-street-pure-bluff-ev) | Two-street EV with independent FE. |
| [`breakevenFoldEquitySecondStreetPureBluff`](/docs/reference/api/fold-equity/breakeven-fold-equity-second-street-pure-bluff) | Breakeven 2nd-street FE given 1st. |
| [`breakevenFoldEquityFirstStreetPureBluff`](/docs/reference/api/fold-equity/breakeven-fold-equity-first-street-pure-bluff) | Breakeven 1st-street FE given 2nd. |
| [`breakevenFoldEquityPureBluffWithAnte`](/docs/reference/api/fold-equity/breakeven-fold-equity-pure-bluff-with-ante) | Pure-bluff fold equity with ante in won pot. |
| [`breakevenFoldEquitySemiBluffWithAnte`](/docs/reference/api/fold-equity/breakeven-fold-equity-semi-bluff-with-ante) | Semi-bluff fold equity with ante. |
| [`twoStreetPureBluffEvWithRake`](/docs/reference/api/fold-equity/two-street-pure-bluff-ev-with-rake) | Two-street air EV with rake. |
| [`threeStreetPureBluffSameFoldEquity`](/docs/reference/api/fold-equity/three-street-pure-bluff-same-fold-equity) | Same fold equity three streets, pure air. |
| [`threeStreetPureBluffEv`](/docs/reference/api/fold-equity/three-street-pure-bluff-ev) | Three-street pure-bluff chip EV. |
