# `breakevenFoldEquityPureBluffWithRake`

Pure-bluff breakeven FE when fold wins `pot+bet` minus rake on shipped pot.

<div className="apiSignature">{`breakevenFoldEquityPureBluffWithRake(potBeforeHeroBet: number, heroBetOrCallSize: number, rakeFraction: number, rakeCap: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Pure bluffs in **raked** games need slightly more fold equity because the pot you steal is reduced by rake on the shipped pot. Parallel to `breakevenFoldEquityPureBluff` with the same rake model as call-EV helpers.

## How to use

```js
const pot = 100;
const bet = 70;
const rakeFraction = 0.05;
const rakeCap = 4;

const fe = poker.breakevenFoldEquityPureBluffWithRake(
  pot,
  bet,
  rakeFraction,
  rakeCap
);
console.log(fe);

console.log(poker.breakevenFoldEquityPureBluff(pot, bet));
```
