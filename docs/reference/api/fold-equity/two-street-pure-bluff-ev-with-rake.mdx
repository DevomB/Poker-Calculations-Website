# `twoStreetPureBluffEvWithRake`

Two-street air EV with rake.

<div className="apiSignature"><code>twoStreetPureBluffEvWithRake(potBeforeStreet1: number, betStreet1: number, betStreet2: number, foldEquityStreet1: number, foldEquityStreet2: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Two-street pure-bluff EV with rake on pots you win when villain folds.

## How to use

```js
const ev = poker.twoStreetPureBluffEvWithRake(
  60, 40, 110,
  0.45, 0.38,
  0.05, 3,
);
const ev0 = poker.twoStreetPureBluffEv(60, 40, 110, 0.45, 0.38);
console.log({ ev, ev0 });
```
