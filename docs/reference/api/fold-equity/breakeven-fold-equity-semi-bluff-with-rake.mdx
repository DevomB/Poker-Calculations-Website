# `breakevenFoldEquitySemiBluffWithRake`

Semi-bluff breakeven fold equity when the **called** pot pays rake via [`rakeFromPot`](/docs/reference/api/pot-and-ev/rake-from-pot) on `totalPotIfCalled`.

<div className="apiSignature"><code>breakevenFoldEquitySemiBluffWithRake(potBeforeHeroBet: number, heroBetSize: number, equityWhenCalled: number, totalPotIfCalled: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Raked **cash** semi-bluff thresholds—same two-outcome model as [`breakevenFoldEquitySemiBluff`](/docs/reference/api/fold-equity/breakeven-fold-equity-semi-bluff), with net called win `equity × (totalPotIfCalled − rake) − heroBetSize`. Higher fold equity than the no-rake version at the same sizes.

## How to use

```js
const poker = require('poker-calculations');

const potBeforeHeroBet = 80;
const heroBetSize = 60;
const equityWhenCalled = 0.32;
const totalPotIfCalled = 200;
const rakeFraction = 0.05;
const rakeCap = 4;

const withRake = poker.breakevenFoldEquitySemiBluffWithRake(
  potBeforeHeroBet,
  heroBetSize,
  equityWhenCalled,
  totalPotIfCalled,
  rakeFraction,
  rakeCap,
);
const noRake = poker.breakevenFoldEquitySemiBluff(
  potBeforeHeroBet,
  heroBetSize,
  equityWhenCalled,
  totalPotIfCalled,
);
console.log({ noRake, withRake, rakeRaisesFe: withRake > noRake });
```

**Returns** `number` — `0` when called EV is already nonnegative; may exceed `1` for −EV lines. Throws on degenerate denominators.
