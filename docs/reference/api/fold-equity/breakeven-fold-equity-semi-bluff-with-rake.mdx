# `breakevenFoldEquitySemiBluffWithRake`

Semi-bluff FE with rake on `totalPotIfCalled`.

<div className="apiSignature">{`breakevenFoldEquitySemiBluffWithRake(potBeforeHeroBet: number, heroBetSize: number, equityWhenCalled: number, totalPotIfCalled: number, rakeFraction: number, rakeCap: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Same semi-bluff model as `breakevenFoldEquitySemiBluff` but **rake is taken from the pot when called** under `rakeFromPot`. Use live-game rake parameters for realistic bluff thresholds.

## How to use

```js
const fe = poker.breakevenFoldEquitySemiBluffWithRake(
  90,
  60,
  0.28,
  90 + 2 * 60,
  0.05,
  5
);
console.log(fe);

const noRake = poker.breakevenFoldEquitySemiBluff(90, 60, 0.28, 210);
console.log('without rake:', noRake);
```
