# `varianceToStandardDeviationPerHand`

`sqrt(variance)` per hand.

<div className="apiSignature"><code>varianceToStandardDeviationPerHand(variancePerHand: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Convert per-hand variance to standard deviation (`sqrt`).

## How to use

```js
console.log(poker.varianceToStandardDeviationPerHand(4)); // 2
```
