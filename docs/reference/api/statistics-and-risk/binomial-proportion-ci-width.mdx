# `binomialProportionCiWidth`

Wilson interval width for a proportion.

<div className="apiSignature"><code>binomialProportionCiWidth(successes: number, nTrials: number, z: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Width of a Wilson (or chosen-z) interval for a binomial proportion — compare to [`wilsonScoreInterval`](/docs/reference/api/statistics-and-risk/wilson-score-interval).

## How to use

```js
const successes = 42;
const nTrials = 100;
const z = 1.96;

const w = poker.binomialProportionCiWidth(successes, nTrials, z);
const interval = poker.wilsonScoreInterval(successes, nTrials, z);
console.log(w, interval.upper - interval.lower);
```
