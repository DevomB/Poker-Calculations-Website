# `binomialProportionCiWidth`

Wilson interval width for a proportion.

<div className="apiSignature"><code>binomialProportionCiWidth(successes: number, nTrials: number, z: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Use when you need wilson interval width for a proportion in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.binomialProportionCiWidth(/* see signature above */);
console.log(result);
```
