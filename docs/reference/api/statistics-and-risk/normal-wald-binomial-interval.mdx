## When to use

Quick **normal approximation** bands when n is moderately large and the proportion is not extremely close to 0 or 1. Prefer `wilsonScoreInterval` or `agrestiCoullInterval` for small n or edge proportions.

## How to use

```js
const poker = require('poker-calculations');

const successes = 250;
const nTrials = 500;
const z = 1.96;

const wald = poker.normalWaldBinomialInterval(successes, nTrials, z);
const wilson = poker.wilsonScoreInterval(successes, nTrials, z);

console.log({ wald, wilson, pHat: successes / nTrials });
```
