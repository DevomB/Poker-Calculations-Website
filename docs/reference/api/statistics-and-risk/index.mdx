# Statistics & risk

Confidence intervals for observed frequencies, Monte Carlo trial sizing, diffusion-style **risk of ruin**, and a Beta–Binomial fold-frequency updater.

Use this group when calibrating sim sample sizes, reporting HUD stats with error bars, or rough bankroll planning—not as a substitute for full stochastic poker modeling.

| Export | Summary |
| --- | --- |
| [`monteCarloStandardError`](./monte-carlo-standard-error) | Binomial SE `√(p̂(1−p̂)/n)`. |
| [`monteCarloTrialsForStandardErrorBound`](./monte-carlo-trials-for-standard-error-bound) | Smallest `n` with SE ≤ target. |
| [`wilsonScoreInterval`](./wilson-score-interval) | Wilson interval `{ lower, upper }`. |
| [`agrestiCoullInterval`](./agresti-coull-interval) | Agresti–Coull interval. |
