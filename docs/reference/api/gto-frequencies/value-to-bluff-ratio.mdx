# `valueToBluffRatio`

`1 / bluffToValueRatio`; `Infinity` when `betSize` is 0.

<div className="apiSignature">{`valueToBluffRatio(potBeforeBet: number, betSize: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Express **how many value combos per bluff** the toy polarized model needs—inverse of `bluffToValueRatio`. Returns `Infinity` when `betSize` is zero (degenerate sizing).

## How to use

```js
const pot = 50;
const bet = 50; // pot-sized bet

console.log(poker.valueToBluffRatio(pot, bet));
console.log(poker.bluffToValueRatio(pot, bet));

console.log(poker.valueToBluffRatio(pot, 0)); // Infinity
```
