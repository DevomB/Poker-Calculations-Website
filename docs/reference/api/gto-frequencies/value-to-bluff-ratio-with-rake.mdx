# `valueToBluffRatioWithRake`

Reciprocal bluff:value with rake.

<div className="apiSignature"><code>valueToBluffRatioWithRake(potBeforeBet: number, betSize: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Reciprocal of [`bluffToValueRatioWithRake`](/docs/reference/api/gto-frequencies/bluff-to-value-ratio-with-rake).

## How to use

```js
const pot = 100;
const bet = 50;
const bluffVal = poker.bluffToValueRatioWithRake(pot, bet, 0.05, 3);
const valBluff = poker.valueToBluffRatioWithRake(pot, bet, 0.05, 3);
console.log(bluffVal * valBluff); // ≈ 1
```
