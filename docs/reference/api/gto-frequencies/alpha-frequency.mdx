# `alphaFrequency`

`1 - minimumDefenseFrequency` = `bet / (pot + bet)`.

<div className="apiSignature">{`alphaFrequency(potBeforeBet: number, betSize: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

**Alpha** is how often villain must fold for a pure bluff to break even if you never get called. Teaching tools pair alpha with estimated fold frequency from population stats.

## How to use

```js
const pot = 80;
const bet = 60;

const alpha = poker.alphaFrequency(pot, bet);
const mdf = poker.minimumDefenseFrequency(pot, bet);
console.log(alpha, mdf, alpha + mdf); // alpha + mdf = 1

const beFe = poker.breakevenFoldEquityPureBluff(pot, bet);
console.log('breakeven FE (pure bluff):', beFe);
```
