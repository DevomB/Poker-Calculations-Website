# GTO-style frequencies

Toy **polarized river** math: minimum defense frequency, alpha (fold weight), and value-to-bluff combo ratios. These are geometry-based teaching helpers—not a full solver.

Minimum defense frequency also appears under [Stacks & display](/docs/reference/api/stacks-and-display/minimum-defense-frequency) for HUD workflows.

| Export | Summary |
| --- | --- |
| [`alphaFrequency`](/docs/reference/api/gto-frequencies/alpha-frequency) | `1 - MDF` = exploit weight if hero never defends. |
| [`bluffToValueRatio`](/docs/reference/api/gto-frequencies/bluff-to-value-ratio) | Polarized river combo ratio. |
| [`valueToBluffRatio`](/docs/reference/api/gto-frequencies/value-to-bluff-ratio) | Reciprocal; `Infinity` when bet is 0. |
