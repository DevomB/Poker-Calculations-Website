# `minimumDefenseFrequencyWithRake`

MDF on post-rake pot geometry.

<div className="apiSignature"><code>minimumDefenseFrequencyWithRake(potBeforeBet: number, betSize: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

MDF on **post-rake** pot geometry after villain bets. At least as high as [`minimumDefenseFrequency`](/docs/reference/api/stacks-and-display/minimum-defense-frequency) without rake.

## How to use

```js
const potBeforeBet = 100;
const betSize = 50;
const rakeFraction = 0.05;
const rakeCap = 3;

const mdf = poker.minimumDefenseFrequencyWithRake(
  potBeforeBet,
  betSize,
  rakeFraction,
  rakeCap,
);
const mdf0 = poker.minimumDefenseFrequency(potBeforeBet, betSize);
console.log({ mdf, mdf0 });
```
