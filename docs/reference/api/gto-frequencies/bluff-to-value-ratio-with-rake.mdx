# `bluffToValueRatioWithRake`

Polarized bluff:value ratio with rake.

<div className="apiSignature"><code>bluffToValueRatioWithRake(potBeforeBet: number, betSize: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Polarized bluff:value ratio when rake shrinks the won pot. At most the no-rake [`bluffToValueRatio`](/docs/reference/api/gto-frequencies/bluff-to-value-ratio).

## How to use

```js
const pot = 100;
const bet = 50;
const rakeFraction = 0.05;
const rakeCap = 3;

const ratio = poker.bluffToValueRatioWithRake(pot, bet, rakeFraction, rakeCap);
console.log(ratio);
```
