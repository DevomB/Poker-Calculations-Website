# `alphaFrequencyWithRake`

`1 − MDF` with rake model.

<div className="apiSignature"><code>alphaFrequencyWithRake(potBeforeBet: number, betSize: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Bluff frequency complement with rake: `1 − MDF_with_rake`.

## How to use

```js
const potBeforeBet = 100;
const betSize = 50;
const rakeFraction = 0.05;
const rakeCap = 3;

const mdf = poker.minimumDefenseFrequencyWithRake(
  potBeforeBet,
  betSize,
  rakeFraction,
  rakeCap,
);
const alpha = poker.alphaFrequencyWithRake(potBeforeBet, betSize, rakeFraction, rakeCap);
console.log(mdf + alpha); // 1
```
