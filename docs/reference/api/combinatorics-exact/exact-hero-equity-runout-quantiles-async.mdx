# `exactHeroEquityRunoutQuantilesAsync`

<div className="apiSignature">{`exactHeroEquityRunoutQuantilesAsync(heroHoleCards, boardCards, range?, options?): Promise<HeroEquityRunoutQuantilesResult>`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Same as [`exactHeroEquityRunoutQuantiles`](/docs/reference/api/combinatorics-exact/exact-hero-equity-runout-quantiles); non-blocking. Optional [`AsyncOptions`](/docs/reference/types/async-options).

## How to use

```js
const poker = require('poker-calculations');

const q = await poker.exactHeroEquityRunoutQuantilesAsync(['Ah', 'Kh'], ['Qh', 'Jh', '2c']);
console.log(q.mean, q.p50);
```

## Return type

[`HeroEquityRunoutQuantilesResult`](/docs/reference/types/hero-equity-runout-quantiles-result).
