# Exact runouts

Enumerate turn/river (or full runouts) for **exact** hero equity spread, nuts/dominated rates, category joints, blocker gradients, and related combinatorics—no Monte Carlo sampling.

Card arguments are [`CardInput`](/docs/concepts/packed-card-input). Optional villain ranges use dense `Float64Array(1326)` or [`SparseRangeSpec`](/docs/reference/types/sparse-range-spec). Async variants accept [`AsyncOptions`](/docs/reference/types/async-options).
