# `exactHeroCategoryAtLeastFlopToRiver`

P(category ≥ threshold) by river.

Card lists use `CardInput` (`string[]` or packed `Uint8Array`). See [Card notation](/docs/concepts/card-notation).

<div className="apiSignature"><code>exactHeroCategoryAtLeastFlopToRiver(heroHoleCards: CardInput, flopThree: CardInput, knownDead: CardInput, minCategoryOrder: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

P(hero best hand ≥ `minCategoryOrder`) by river. Category order matches [`handRankCategoryOrder`](/docs/reference/api/hands-and-equity/hand-rank-category-order).

## How to use

```js
const hole = ['Ah', 'Kh'];
const flop = ['2h', '7d', '9c'];
const minCategoryOrder = 6; // flush tier — see handRankCategoryOrder

const p = poker.exactHeroCategoryAtLeastFlopToRiver(hole, flop, [], minCategoryOrder);
console.log(p);
```
