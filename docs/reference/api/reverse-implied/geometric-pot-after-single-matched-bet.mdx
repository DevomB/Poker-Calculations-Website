# `geometricPotAfterSingleMatchedBet`

Pot after one matched bet/raise.

<div className="apiSignature"><code>geometricPotAfterSingleMatchedBet(pot0: number, betSize: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Pot after **one** matched bet/raise: `pot0 + 2×betSize`. Single step of [`geometricPotAfterMatchedPotFractions`](/docs/reference/api/reverse-implied/geometric-pot-after-matched-pot-fractions).

## How to use

```js
const pot0 = 100;
const betSize = 40;

console.log(poker.geometricPotAfterSingleMatchedBet(pot0, betSize)); // 180
```
