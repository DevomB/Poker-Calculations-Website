# Reverse implied & geometry

Toy models for **reverse implied odds** (how much extra you can lose when behind) and **geometric pot growth** when both players keep betting fixed pot fractions heads-up.

Use these when teaching implied vs reverse implied odds or illustrating how a pot balloons over multiple streets of matched sizing—not as a full game-tree solver.

| --- | --- |
| [`reverseImpliedOddsMaxFutureLoss`](./reverse-implied-odds-max-future-loss) | Toy ceiling on extra future loss when losing. |
| [`geometricPotAfterMatchedPotFractions`](./geometric-pot-after-matched-pot-fractions) | Pot after `n` matched pot-fraction HU rounds. |
