# Reverse implied & geometry

Toy models for **reverse implied odds** (how much extra you can lose when behind) and **geometric pot growth** when both players keep betting fixed pot fractions heads-up.

Use these when teaching implied vs reverse implied odds or illustrating how a pot balloons over multiple streets of matched sizing—not as a full game-tree solver.

| Export | Summary |
| --- | --- |
| [`reverseImpliedOddsMaxFutureLoss`](/docs/reference/api/reverse-implied/reverse-implied-odds-max-future-loss) | Toy ceiling on extra future loss when losing. |
| [`geometricPotAfterMatchedPotFractions`](/docs/reference/api/reverse-implied/geometric-pot-after-matched-pot-fractions) | Pot after `n` matched pot-fraction HU rounds. |
| [`reverseImpliedOddsMinEquity`](/docs/reference/api/reverse-implied/reverse-implied-odds-min-equity) | Minimum equity for reverse-implied bound. |
| [`geometricPotAfterSingleMatchedBet`](/docs/reference/api/reverse-implied/geometric-pot-after-single-matched-bet) | Pot after one matched bet/raise. |
