# `reverseImpliedOddsMinEquity`

Minimum equity for reverse-implied bound.

<div className="apiSignature"><code>reverseImpliedOddsMinEquity(potBeforeCall: number, toCall: number, maxFutureLoss: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Minimum equity for a call when you cap future **losses** at `maxFutureLoss` — inverse of [`reverseImpliedOddsMaxFutureLoss`](/docs/reference/api/reverse-implied/reverse-implied-odds-max-future-loss).

## How to use

```js
const potBeforeCall = 100;
const toCall = 50;
const maxFutureLoss = 200;

const minEq = poker.reverseImpliedOddsMinEquity(potBeforeCall, toCall, maxFutureLoss);
const maxLoss = poker.reverseImpliedOddsMaxFutureLoss(potBeforeCall, toCall, minEq);
console.log(maxLoss); // ≈ maxFutureLoss
```
