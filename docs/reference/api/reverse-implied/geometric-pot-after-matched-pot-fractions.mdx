# `geometricPotAfterMatchedPotFractions`

Pot after `nRounds` of matched pot-fraction betting heads-up.

<div className="apiSignature">{`geometricPotAfterMatchedPotFractions(pot0: number, fraction: number, nRounds: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Model **geometric pot growth** when both players bet the same fraction of the pot each street (`pot0 × (1 + 2f)^n`). Useful for illustrating why deep stacks balloon pots and for planning stack-off lines.

## How to use

```js
const startPot = 10;
const betFraction = 0.75; // each player bets 0.75× pot per round, both call
const rounds = 3;

const pot = poker.geometricPotAfterMatchedPotFractions(startPot, betFraction, rounds);
console.log(pot); // 10 * (1 + 2*0.75)^3 = 10 * 2.5^3 = 156.25
```
