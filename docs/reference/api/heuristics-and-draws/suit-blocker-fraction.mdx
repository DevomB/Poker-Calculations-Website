# `suitBlockerFraction`

Dead suit cards divided by unseen.

<div className="apiSignature"><code>suitBlockerFraction(suitCardsDead: number, unseen: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Fraction of unseen cards that are dead in a given suit — input to [`blockerAdjustedOuts`](/docs/reference/api/heuristics-and-draws/blocker-adjusted-outs).

## How to use

```js
const suitCardsDead = 3; // e.g. two hearts in villain range + one on board
const unseen = 44;

const frac = poker.suitBlockerFraction(suitCardsDead, unseen);
console.log(frac); // 3/44
```
