# `flopToTurnAtLeastOneHitUnionTwoCategories`

Two overlapping categories, one street (flop→turn).

<div className="apiSignature"><code>flopToTurnAtLeastOneHitUnionTwoCategories(unseen: number, outsA: number, outsB: number, sharedAb: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Flop→turn hit rate when two out categories overlap (e.g. flush + straight). Union size is `outsA + outsB - sharedAb` before the one-street formula.

## How to use

```js
const unseen = 47;
const flushOuts = 9;
const straightOuts = 4;
const shared = 1; // one card counts toward both

const p = poker.flopToTurnAtLeastOneHitUnionTwoCategories(
  unseen,
  flushOuts,
  straightOuts,
  shared,
);
console.log(p); // 12/47 — same union logic as flop→river, one card only
```
