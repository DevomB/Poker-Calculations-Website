# `turnToRiverAtLeastOneHitDisjointOutsSum`

Sum disjoint outs, turn→river.

<div className="apiSignature"><code>turnToRiverAtLeastOneHitDisjointOutsSum(unseen: number, outsPerCategory: number[]): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Sum disjoint outs on turn→river (one card).

## How to use

```js
const p = poker.turnToRiverAtLeastOneHitDisjointOutsSum(44, [9, 4]);
console.log(p); // 13/44
```
