# `turnToRiverAtLeastOneHitProbability`

One-street turn→river hit: `outs/unseen`.

<div className="apiSignature"><code>turnToRiverAtLeastOneHitProbability(outs: number, unseenAfterTurn: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

One card to come on the turn (river only). Use when only the river remains and you have a single disjoint out count.

## How to use

```js
const outs = 9;
const unseen = 44; // after turn (2 hole + 4 board)

const pRiver = poker.turnToRiverAtLeastOneHitProbability(outs, unseen);
console.log(pRiver); // 9/44 ≈ 0.205
```
