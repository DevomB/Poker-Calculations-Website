# Heuristics & draws

Fast equity estimates (rule of 2/4), hypergeometric and flop→river hit math, runner–runner patterns, implied-odds breakeven targets, and multiway out duplication. Exact straight-on-river probability is under [Monte Carlo](/docs/reference/api/monte-carlo/) (`straightMadeFlopToRiverExactProbability`).

Use this group in draw trainers and study apps where exact enumeration is overkill but you need principled probabilities from out counts and unseen cards.
