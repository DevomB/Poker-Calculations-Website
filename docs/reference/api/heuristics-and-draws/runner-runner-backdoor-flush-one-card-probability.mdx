# `runnerRunnerBackdoorFlushOneCardProbability`

One-card backdoor flush probability.

<div className="apiSignature"><code>runnerRunnerBackdoorFlushOneCardProbability(suitCardsRemaining: number, unseenCards: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

One-card backdoor flush probability: `suitRemaining / unseen`. Upper bound on the two-card runner model when only one street remains.

## How to use

```js
const suitRemaining = 10; // hearts left in deck
const unseen = 47;

const pOne = poker.runnerRunnerBackdoorFlushOneCardProbability(suitRemaining, unseen);
const pTwo = poker.runnerRunnerBackdoorFlushTwoCardProbability(suitRemaining, unseen);
console.log({ pOne, pTwo }); // pOne <= pTwo
```
