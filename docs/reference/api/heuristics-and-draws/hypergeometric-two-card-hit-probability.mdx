# `hypergeometricTwoCardHitProbability`

Two-card hypergeometric hit rate.

<div className="apiSignature"><code>hypergeometricTwoCardHitProbability(outs: number, unseenCards: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Exact two-card hit rate without enumerating runouts. Equivalent to [`flopToRiverAtLeastOneHitProbability`](/docs/reference/api/heuristics-and-draws/flop-to-river-at-least-one-hit-probability) for the same `outs` and `unseen`.

## How to use

```js
const outs = 9;
const unseen = 47;

const pTwo = poker.hypergeometricTwoCardHitProbability(outs, unseen);
const pFlopRiver = poker.flopToRiverAtLeastOneHitProbability(outs, unseen);
console.log(pTwo, pFlopRiver); // equal
```
