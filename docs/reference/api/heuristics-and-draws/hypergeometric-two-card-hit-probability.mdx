# `hypergeometricTwoCardHitProbability`

Two-card hypergeometric hit rate.

<div className="apiSignature"><code>hypergeometricTwoCardHitProbability(outs: number, unseenCards: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Use when you need two-card hypergeometric hit rate in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.hypergeometricTwoCardHitProbability(/* see signature above */);
console.log(result);
```
