# `hypergeometricTwoCardMissProbability`

Complement of two-card hit.

<div className="apiSignature"><code>hypergeometricTwoCardMissProbability(outs: number, unseenCards: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Complement of two-card hit — useful for miss rates and sanity checks (`hit + miss === 1`).

## How to use

```js
const outs = 9;
const unseen = 47;

const hit = poker.hypergeometricTwoCardHitProbability(outs, unseen);
const miss = poker.hypergeometricTwoCardMissProbability(outs, unseen);
console.log(hit + miss); // 1
```
