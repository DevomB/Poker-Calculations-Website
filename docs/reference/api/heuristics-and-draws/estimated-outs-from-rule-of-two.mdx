# `estimatedOutsFromRuleOfTwo`

Uncapped inverse-style estimate `equityﾃ蓉nseen/2` clamped to `[0, unseen]` (not exact inverse of capped rule-of-two).

<div className="apiSignature"><code>estimatedOutsFromRuleOfTwo(equity: number, unseenCards: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Back-solve approximate out counts from a one-street equity target窶排ough inverse of the rule-of-two line, not a perfect inverse of capped [`ruleOfTwoEquity`](/docs/reference/api/heuristics-and-draws/rule-of-two-equity).

## How to use

```js
const poker = require('poker-calculations');

const outs = poker.estimatedOutsFromRuleOfTwo(0.32, 47);
console.log(outs);
```
