# `blockerAdjustedOuts`

Scale outs by blocker fraction.

<div className="apiSignature"><code>blockerAdjustedOuts(outs: number, blockerFraction: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Scale a raw out count down when villains block part of your outs (`outs × (1 - blockerFraction)`), clamped to `[0, outs]`.

## How to use

```js
const rawOuts = 9;
const blockerFraction = 0.15; // ~15% of outs blocked

const adj = poker.blockerAdjustedOuts(rawOuts, blockerFraction);
console.log(adj); // 9 * 0.85 = 7.65
```
