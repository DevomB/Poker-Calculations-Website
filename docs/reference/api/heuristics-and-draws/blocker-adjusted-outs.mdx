# `blockerAdjustedOuts`

Scale outs by blocker fraction.

<div className="apiSignature"><code>blockerAdjustedOuts(outs: number, blockerFraction: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Use when you need scale outs by blocker fraction in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.blockerAdjustedOuts(/* see signature above */);
console.log(result);
```
