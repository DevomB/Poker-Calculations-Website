# `impliedOddsRequiredEquityFromFutureWin`

Equity needed given extra future win.

<div className="apiSignature"><code>impliedOddsRequiredEquityFromFutureWin(potBeforeCall: number, toCall: number, futureWin: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Inverse of [`impliedBreakevenFutureWin`](/docs/reference/api/heuristics-and-draws/implied-breakeven-future-win): equity needed when you expect a fixed extra win on later streets.

## How to use

```js
const potBeforeCall = 80;
const toCall = 40;
const futureWin = 120;

const need = poker.impliedOddsRequiredEquityFromFutureWin(potBeforeCall, toCall, futureWin);
console.log(need);

const future = poker.impliedBreakevenFutureWin(potBeforeCall, toCall, need);
console.log(future); // ≈ futureWin
```
