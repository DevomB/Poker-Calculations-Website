# `impliedBreakevenTotalPot`

Total pot needed: `(pot+call)/equity`.

<div className="apiSignature"><code>impliedBreakevenTotalPot(potBeforeCall: number, toCall: number, equity: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Total pot size (before your call is included in the numerator) needed for a neutral call at given equity: `(potBefore + toCall) / equity`.

## How to use

```js
const potBeforeCall = 80;
const toCall = 40;
const equity = 0.25;

const totalPot = poker.impliedBreakevenTotalPot(potBeforeCall, toCall, equity);
console.log(totalPot); // (80+40)/0.25 = 480

// Compare to extra future win at same equity
console.log(poker.impliedBreakevenFutureWin(potBeforeCall, toCall, equity));
```
