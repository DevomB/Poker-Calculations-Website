# `flopToTurnAtLeastOneHitUnionThreeCategories`

Three categories with inclusion–exclusion, one street.

<div className="apiSignature"><code>flopToTurnAtLeastOneHitUnionThreeCategories(unseen: number, outsA: number, outsB: number, outsC: number, sharedAb: number, sharedAc: number, sharedBc: number, sharedAbc: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Three overlapping out families on flop→turn. Pass pairwise and triple **shared card counts**; inclusion–exclusion runs before the one-street hit formula.

## How to use

```js
// Mostly disjoint toy categories on the turn
const p = poker.flopToTurnAtLeastOneHitUnionThreeCategories(
  47,
  9, 4, 2, // outs A, B, C
  0, 0, 0, // shared AB, AC, BC
  0,       // shared ABC
);
console.log(p);
```
