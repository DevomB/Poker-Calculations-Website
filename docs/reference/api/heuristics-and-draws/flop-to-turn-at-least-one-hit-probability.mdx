# `flopToTurnAtLeastOneHitProbability`

One-street flop→turn hit: `outs/unseen`.

<div className="apiSignature"><code>flopToTurnAtLeastOneHitProbability(outs: number, unseenAfterFlop: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

One card to come on the flop (turn only). Same hypergeometric rate as [`hypergeometricOneCardHitProbability`](/docs/reference/api/heuristics-and-draws/hypergeometric-one-card-hit-probability) but named for the flop→turn street.

## How to use

```js
const outs = 9;
const unseen = 47; // deck after flop (2 hole + 3 flop)

const pTurn = poker.flopToTurnAtLeastOneHitProbability(outs, unseen);
console.log(pTurn); // 9/47 ≈ 0.191

// Compare to full flop→river two-card draw
console.log(poker.flopToRiverAtLeastOneHitProbability(outs, unseen));
```
