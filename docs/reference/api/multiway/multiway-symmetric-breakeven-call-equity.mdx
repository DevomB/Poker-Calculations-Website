# `multiwaySymmetricBreakevenCallEquity`

Symmetric extra callers each matching `toCall`.

<div className="apiSignature">{`multiwaySymmetricBreakevenCallEquity(potBefore: number, toCall: number, symmetricExtraCallers: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

You face a call and **k additional villains** each call the same amount (symmetric cold-call geometry). Breakeven equity rises vs heads-up because you must win a larger final pot for the same call price.

## How to use

```js
const pot = 60;
const toCall = 30;
const extraCallers = 2; // two callers behind you

const beHu = poker.breakevenCallEquity(pot, toCall);
const beMw = poker.multiwaySymmetricBreakevenCallEquity(pot, toCall, extraCallers);
console.log('HU breakeven:', beHu);
console.log('3-way breakeven:', beMw); // higher than HU
```
