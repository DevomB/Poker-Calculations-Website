# Multiway

Symmetric **multiway call** breakeven equity when extra callers each match the hero’s call size, with optional pot-share when hero wins.

Chip breakeven only—combine with duplicated-out heuristics and sim equity for full multiway spots.

| Export | Summary |
| --- | --- |
| [`multiwaySymmetricBreakevenCallEquity`](/docs/reference/api/multiway/multiway-symmetric-breakeven-call-equity) | `k` extra symmetric callers. |
| [`multiwaySymmetricBreakevenCallEquityWithShare`](/docs/reference/api/multiway/multiway-symmetric-breakeven-call-equity-with-share) | Winner-take-all or fractional pot share. |
| [`multiwaySymmetricBreakevenCallEquityWithRake`](/docs/reference/api/multiway/multiway-symmetric-breakeven-call-equity-with-rake) | Multiway breakeven call equity with rake. |
| [`multiwaySymmetricBreakevenCallEquityWithShareAndRake`](/docs/reference/api/multiway/multiway-symmetric-breakeven-call-equity-with-share-and-rake) | Share model + rake on multiway call. |
| [`multiwayExpectedValueCall`](/docs/reference/api/multiway/multiway-expected-value-call) | Call EV with symmetric extra callers. |
