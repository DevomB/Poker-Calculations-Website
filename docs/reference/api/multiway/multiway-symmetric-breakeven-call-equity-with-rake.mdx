# `multiwaySymmetricBreakevenCallEquityWithRake`

Multiway breakeven call equity with rake.

<div className="apiSignature"><code>multiwaySymmetricBreakevenCallEquityWithRake(potBefore: number, toCall: number, symmetricExtraCallers: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Symmetric multiway call breakeven with rake on the final pot. At least [`multiwaySymmetricBreakevenCallEquity`](/docs/reference/api/multiway/multiway-symmetric-breakeven-call-equity) without rake.

## How to use

```js
const pot = 60;
const toCall = 30;
const k = 2;
const rakeFraction = 0.05;
const rakeCap = 3;

const be = poker.multiwaySymmetricBreakevenCallEquityWithRake(
  pot, toCall, k, rakeFraction, rakeCap,
);
console.log(be);
```
