# `multiwaySymmetricBreakevenCallEquityWithShare`

Same pot geometry with explicit pot-share when hero wins.

<div className="apiSignature">{`multiwaySymmetricBreakevenCallEquityWithShare(potBefore: number, toCall: number, symmetricExtraCallers: number, shareModel: 0 | 1, heroFractionWhenWin: number): number`}</div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Extend symmetric multiway breakeven when hero does not take **100% of the pot** when winning—`shareModel` `0` is winner-take-all; `1` multiplies the final pot by `heroFractionWhenWin` (chop proxy, side-pot share, etc.).

## How to use

```js
const pot = 100;
const toCall = 40;
const extra = 1;

const wta = poker.multiwaySymmetricBreakevenCallEquityWithShare(
  pot, toCall, extra, 0, 1
);
const chop = poker.multiwaySymmetricBreakevenCallEquityWithShare(
  pot, toCall, extra, 1, 0.5
);
console.log('winner-take-all BE:', wta);
console.log('50% share when win BE:', chop); // higher equity needed
```
