# `multiwaySymmetricBreakevenCallEquityWithShareAndRake`

Share model + rake on multiway call.

<div className="apiSignature"><code>multiwaySymmetricBreakevenCallEquityWithShareAndRake(potBefore: number, toCall: number, symmetricExtraCallers: number, shareModel: 0 | 1, heroFractionWhenWin: number, rakeFraction: number, rakeCap: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Multiway breakeven with [`Multiway_symmetric_pot_share_model`](/docs/reference/api/multiway/multiway-symmetric-breakeven-call-equity-with-share) and rake.

## How to use

```js
const need = poker.multiwaySymmetricBreakevenCallEquityWithShareAndRake(
  90, 30, 2,
  0,    // 0 = winner-takes-all share model
  1.0,  // hero fraction when win (ignored for model 0)
  0.05, 3,
);
console.log(need);
```
