# `multiwayExpectedValueCall`

Call EV with symmetric extra callers.

<div className="apiSignature"><code>multiwayExpectedValueCall(equity: number, potBefore: number, toCall: number, symmetricExtraCallers: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Call chip EV with `k` symmetric extra callers each matching `toCall`. Reduces to [`expectedValueCall`](/docs/reference/api/pot-and-ev/expected-value-call) when `k = 0`.

## How to use

```js
const equity = 0.4;
const pot = 60;
const toCall = 30;

const evHu = poker.expectedValueCall(equity, pot, toCall);
const evMw = poker.multiwayExpectedValueCall(equity, pot, toCall, 2);
console.log({ evHu, evMw });
```
