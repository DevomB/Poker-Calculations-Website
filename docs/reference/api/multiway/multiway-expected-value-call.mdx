# `multiwayExpectedValueCall`

Call EV with symmetric extra callers.

<div className="apiSignature"><code>multiwayExpectedValueCall(equity: number, potBefore: number, toCall: number, symmetricExtraCallers: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Use when you need call ev with symmetric extra callers in trainers, study tools, or backend scripts. Values follow the native C++ implementation in `poker-calculations`.

## How to use

```js
const poker = require('poker-calculations');

const result = poker.multiwayExpectedValueCall(/* see signature above */);
console.log(result);
```
