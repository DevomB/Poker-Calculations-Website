# Stacks & display

Stack geometry (SPR, effective stacks, big-blind sizing), human-readable pot odds, tournament **M** and **Q**, orbit cost, minimum raise sizing, and preflop combo counting. Chip-share normalization here is **not** Harville ICM—that lives under [ICM](/docs/reference/api/icm/).

Use these exports for HUD labels, tournament push/fold charts, and preflop range math where you need fast display helpers without running a full sim.

| Export | Summary |
| --- | --- |
| [`spr`](./spr) | Stack-to-pot ratio. |
| [`effectiveStack`](./effective-stack) | Minimum stack; empty → `0`. |
| [`normalizedStackFractions`](./normalized-stack-fractions) | Each stack / sum(stacks) (chip shares). |
| [`minimumDefenseFrequency`](./minimum-defense-frequency) | MDF from pot geometry. |
| [`stackInBigBlinds`](./stack-in-big-blinds) | Stack size in big blinds. |
| [`potOddsRatioDisplay`](./pot-odds-ratio-display) | Display ratio `pot : to_call`. |
| [`formatPotOdds`](./format-pot-odds) | Human-readable `"x:1"` string. |
| [`harringtonM`](./harrington-m) | Harrington **M** = stack / (sb + bb + antes). |
| [`harringtonMEffective`](./harrington-m-effective) | Effective M with active antes. |
| [`harringtonMEffectiveActiveAntes`](./harrington-m-effective-active-antes) | Effective M with per-seat antes. |
| [`harringtonQ`](./harrington-q) | Hero stack / mean(table stacks). |
| [`orbitCostChips`](./orbit-cost-chips) | One orbit: sb + bb + sum(antes). |
| [`nlMinimumRaiseToTotal`](./nl-minimum-raise-to-total) | NL minimum **total** wager after a raise. |
| [`preflopCombosFromNotation`](./preflop-combos-from-notation) | Combo count from `AA`, `AKs`, `AKo`. |
| [`preflopCombosFromNotationsList`](./preflop-combos-from-notations-list) | Sum of combo counts over a list. |

Rake-aware and chip-EV breakeven helpers are under [Pot & EV](/docs/reference/api/pot-and-ev/).
