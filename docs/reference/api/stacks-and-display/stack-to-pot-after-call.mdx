# `stackToPotAfterCall`

Stack:pot ratio after call (inverse SPR).

<div className="apiSignature"><code>stackToPotAfterCall(potBeforeCall: number, toCall: number, effectiveStackBeforeCall: number): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Stack-to-pot ratio after a call — inverse of [`sprAfterCall`](/docs/reference/api/sizing-and-commitment/spr-after-call).

## How to use

```js
const potBeforeCall = 100;
const toCall = 50;
const stack = 300;

const spr = poker.sprAfterCall(potBeforeCall, toCall, stack);
const stp = poker.stackToPotAfterCall(potBeforeCall, toCall, stack);
console.log(spr * stp); // ≈ 1
```
