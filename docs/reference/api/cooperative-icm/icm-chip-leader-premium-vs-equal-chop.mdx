# `icmChipLeaderPremiumVsEqualChop`

Chip leader surplus ratio vs equal chop.

<div className="apiSignature"><code>icmChipLeaderPremiumVsEqualChop(stacks: F64VectorInput, payouts: F64VectorInput): number</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Chip leader ICM surplus vs equal chop, expressed as a ratio minus 1.

## How to use

```js
const stacks = [15000, 5000, 3000, 2000];
const payouts = [5000, 3000, 1500, 500];

console.log(poker.icmChipLeaderPremiumVsEqualChop(stacks, payouts));
```
