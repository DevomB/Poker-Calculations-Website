# `icmShapleyValues`

<div className="apiSignature">{`icmShapleyValues(stacks, payouts, options?): IcmShapleyValuesResult`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Negotiating a tournament chop or comparing fair-share splits to Harville ICM $EV.

More walkthroughs: [Tournament, runouts, and bots](/docs/guides/tournament-runouts-and-bots).

## How to use

```js
const poker = require('poker-calculations');

const stacks = [12000, 8000, 5000, 2000];
const payouts = [10000, 6000, 3000, 1000];

const shapley = poker.icmShapleyValues(stacks, payouts, { method: 'exact' });
console.log(shapley.values);
```

## Return type

[`IcmShapleyValuesResult`](/docs/reference/types/icm-shapley-values-result) — optional [`IcmShapleyValuesOptions`](/docs/reference/types/icm-shapley-values-options).
