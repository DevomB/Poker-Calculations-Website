# `icmChopNegotiationAnalysis`

<div className="apiSignature">{`icmChopNegotiationAnalysis(stacks, payouts): IcmChopNegotiationAnalysisResult`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Compare chip-proportional chop dollars to Harville ICM and list feasible surplus transfers between seats.

## How to use

```js
const poker = require('poker-calculations');

const analysis = poker.icmChopNegotiationAnalysis(
  [12000, 8000, 5000, 2000],
  [10000, 6000, 3000, 1000],
);
console.log(analysis.surplus, analysis.paretoPairs);
```

[`IcmChopNegotiationAnalysisResult`](/docs/reference/types/icm-chop-negotiation-analysis-result) — nested [`IcmChopParetoPair`](/docs/reference/types/icm-chop-pareto-pair).
