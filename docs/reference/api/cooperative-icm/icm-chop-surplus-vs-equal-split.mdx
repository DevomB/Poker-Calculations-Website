# `icmChopSurplusVsEqualSplit`

ICM EV minus equal chop per seat.

<div className="apiSignature"><code>icmChopSurplusVsEqualSplit(stacks: F64VectorInput, payouts: F64VectorInput, returnFormat?: F64ReturnFormat): number[] | Float64Array</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

Per-seat ICM EV minus equal chop — who gains/loses vs splitting the pool evenly.

## How to use

```js
const stacks = [10000, 7500, 5000, 2500];
const payouts = [5000, 3000, 1500, 500];

const surplus = poker.icmChopSurplusVsEqualSplit(stacks, payouts);
console.log(surplus.reduce((a, b) => a + b, 0)); // ≈ 0
```
