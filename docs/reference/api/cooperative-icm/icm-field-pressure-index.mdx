# `icmFieldPressureIndex`

<div className="apiSignature">{`icmFieldPressureIndex(stacks, payouts, heroIndex, potChips): IcmFieldPressureIndexResult`}</div>

## Import

```js
const poker = require('poker-calculations');
```

## When to use

Summarize bubble pressure on the hero against the remaining field at a given pot size.

## How to use

```js
const poker = require('poker-calculations');

const r = poker.icmFieldPressureIndex(
  [12000, 8000, 5000, 2000],
  [10000, 6000, 3000, 1000],
  0,
  500,
);
console.log(r.index, r.argmaxVillain);
```
