# `icmExpectedPayoutsDeltaFromChipChop`

ICM minus chip-proportional chop.

<div className="apiSignature"><code>icmExpectedPayoutsDeltaFromChipChop(stacks: F64VectorInput, payouts: F64VectorInput, returnFormat?: F64ReturnFormat): number[] | Float64Array</code></div>

## Import

```js
const poker = require('poker-calculations');
```

ESM (Node):

```js
import { createRequire } from 'module';
const require = createRequire(import.meta.url);
const poker = require('poker-calculations');
```

## When to use

ICM expected payouts minus a **chip-proportional** chop for each seat.

## How to use

```js
const stacks = [12000, 8000, 5000, 2000];
const payouts = [10000, 6000, 3000, 1000];

const delta = poker.icmExpectedPayoutsDeltaFromChipChop(stacks, payouts);
console.log(delta.reduce((a, b) => a + b, 0)); // ≈ 0
```
