# TypeScript types

Shapes exported from `poker-calculations` for `decideAction`, Monte Carlo batches, ICM vectors, and hand evaluation. Import types with:

```ts
import type { CardInput, NativePokerState } from 'poker-calculations';
```

## Result and state objects

| Type | Doc |
| --- | --- |
| [`NativePokerState`](/docs/reference/types/native-poker-state) | Table snapshot for `decideAction` |
| [`NativeBotConfig`](/docs/reference/types/native-bot-config) | Bot tuning |
| [`NativeOpponentModel`](/docs/reference/types/native-opponent-model) | Optional villain model |
| [`DecisionResult`](/docs/reference/types/decision-result) | `decideAction` output |
| [`HandEvalResult`](/docs/reference/types/hand-eval-result) | Full `evaluateBestHand` result |
| [`HandEvalResultSlim`](/docs/reference/types/hand-eval-result-slim) | Slim `evaluateBestHand` result |
| [`EvaluatorBenchmarkResult`](/docs/reference/types/evaluator-benchmark-result) | Evaluator benchmark |
| [`McEquityDetailedResult`](/docs/reference/types/mc-equity-detailed-result) | Detailed MC + CI |
| [`WilsonScoreInterval`](/docs/reference/types/wilson-score-interval) | Interval bounds |
| [`BetaBinomialFoldPosterior`](/docs/reference/types/beta-binomial-fold-posterior) | Fold-rate posterior |
| [`SidePotLayer`](/docs/reference/types/side-pot-layer) | Side-pot layer |
