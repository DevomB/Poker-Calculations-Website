# `McEquityDetailedResult`

Returned by [`simulateHandOutcomeDetailed`](/docs/reference/api/monte-carlo/simulate-hand-outcome-detailed).

| Field | Type | Meaning |
| --- | --- | --- |
| `estimate` | `number` | Mean equity share |
| `se` | `number` | Monte Carlo standard error |
| `ciLow` | `number` | Wilson interval low |
| `ciHigh` | `number` | Wilson interval high |
| `n` | `number` | Trials run |

Wilson bounds use [`wilsonScoreInterval`](/docs/reference/api/statistics-and-risk/wilson-score-interval) with `successes = round(estimate × n)`. See [Numerical semantics](/docs/concepts/numerical-semantics#detailed-monte-carlo-intervals).
