# ICM and side pots

## ICM (Harville)

Start with [`icmWinProbabilitiesHarville`](/docs/reference/api/icm/icm-win-probabilities-harville) and [`icmExpectedPayouts`](/docs/reference/api/icm/icm-expected-payouts). Placement matrices and bubble factors are under the [ICM API category](/docs/reference/api/icm/).

For chip-utility ICM (independent Weitzman model, `stack^alpha`), use [`icmExpectedPayoutsWeitzman`](/docs/reference/api/icm/icm-expected-payouts-weitzman) instead of Harville placement math.

## Side pots

[`sidePotLadderFromCommitments`](/docs/reference/api/side-pots/side-pot-ladder-from-commitments) builds layers; [`layeredPotChipEvFromEquities`](/docs/reference/api/side-pots/layered-pot-chip-ev-from-equities) assigns chip EV per player per layer.
