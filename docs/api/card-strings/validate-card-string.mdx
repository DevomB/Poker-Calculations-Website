## When to use

Use these at the boundary of any pipeline that accepts pasted hands, solver exports, or UI text. They keep card identity consistent before you spend CPU on evaluation or simulation.

**Concrete trigger:** `true` if `card` parses as one card (`Ah`, `10c`, …).

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
