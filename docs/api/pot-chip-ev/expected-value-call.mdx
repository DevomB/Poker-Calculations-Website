## When to use

Use for single-street chip EV and breakeven math around calls, pot odds, rake on the final pot, and book-style odds conversion—ideal for study sheets and post-hand EV checks.

**Concrete trigger:** Chip EV of calling once vs folding (0); no future streets.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
