## When to use

Use after you know each player’s committed chips for the hand: build side-pot layers, sum chip EV across layers from per-layer equities, or total chips across layers.

**Concrete trigger:** Sum of `potChips` across layers from `sidePotLadderFromCommitments`.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
