## When to use

Use these once cards are validated: resolving best five from up to seven, comparing spots, or bucketing into categories for reports. They are deterministic and do not depend on stack or pot.

**Concrete trigger:** Encoded strength: decimal string of `uint64` (rank in high bits + five kicker nibbles); comparable across spots.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
