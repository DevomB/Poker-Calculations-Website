## When to use

Use for SPR, blinds-relative stack metrics, Harrington M/Q, orbit cost, NL minimum raise toy sizing, and human-readable pot-odds strings—anything that turns chips into ratios for HUDs or notes.

**Concrete trigger:** Each stack divided by sum of stacks (chip shares; not Harville ICM).

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
