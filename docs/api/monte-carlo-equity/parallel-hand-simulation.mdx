## When to use

Use when you need equity estimates against random or uniform villains—either fast single-threaded sampling or threaded batches, or exact HU numbers on a fixed board.

**Concrete trigger:** Same with worker threads and distinct seeds per chunk.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
