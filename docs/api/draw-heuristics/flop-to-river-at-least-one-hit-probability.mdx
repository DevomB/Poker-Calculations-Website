## When to use

Use for quick draw math: rule of 2/4, flop-to-union hit models, runner-runner patterns, duplication heuristics, and exact straight-or-better flop-to-river probabilities when you already modeled outs.

**Concrete trigger:** Two streets, single effective out count, no hit on both misses.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
