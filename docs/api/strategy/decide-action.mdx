## Summary

Native export `decideAction` from the `poker-calculations` addon.

## Signature

```ts
decideAction(
    state: NativePokerState,
    config: NativeBotConfig,
    opponentModel?: NativeOpponentModel | null,
    heroSeat?: number
  ): DecisionResult;
```

## Implementation notes

```text
_See TypeScript types in the repository._
```

## When to use

Use when you want a single rule-based action from a serialized table state (bots, trainers, regression fixtures)—not when you need full GTO solutions.

**Concrete trigger:** See the summary below.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.

const poker = require('poker-calculations');
// decideAction(...) — see signature above
