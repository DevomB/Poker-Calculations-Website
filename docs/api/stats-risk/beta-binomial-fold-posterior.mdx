## When to use

Use when you are quantifying uncertainty or risk around estimates: MC standard errors, confidence intervals, Hoeffding trial counts, diffusion-style risk of ruin, or Beta updates on observed folds.

**Concrete trigger:** Conjugate Beta update; returns `{ alpha, beta, posteriorMean }`.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addonŌĆöthis library is Node + N-API.
