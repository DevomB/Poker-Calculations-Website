## Summary

Native export `breakevenFoldEquityPureBluff` from the `poker-calculations` addon.

## Signature

```ts
breakevenFoldEquityPureBluff(potBeforeHeroBet: number, heroBetOrCallSize: number): number;
```

## Implementation notes

```text
_See TypeScript types in the repository._
```

## When to use

Use when you are sizing bluffs or semi-bluffs and need breakeven fold equity, optionally with rake or across two streets with independent fold assumptions.

**Concrete trigger:** See the summary below.

**Skip this export** when you need full multi-street game trees, opponent-specific ranges, or browser-side execution without the native addon—this library is Node + N-API.
