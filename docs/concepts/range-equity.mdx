# Range equity

Range equity is the weighted average of heads-up outcomes over villain hole-card combos with blockers removed.

Use [`exactHuEquityVsRange`](/docs/reference/api/hands-and-equity/exact-hu-equity-vs-range) for exact enumeration and [`simulateEquityVsRange`](/docs/reference/api/monte-carlo/simulate-equity-vs-range) when Monte Carlo is enough.

See [Sparse range input](/docs/concepts/sparse-range-input) for the `{ indices, weights }` shape.
