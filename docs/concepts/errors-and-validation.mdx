# Errors and validation

Invalid or duplicate cards throw `Error` from native code. Overlapping card sets between two hands throw on [`compareBestHands`](/docs/reference/api/hands-and-equity/compare-best-hands).

Use [`validateCardString`](/docs/reference/api/hands-and-equity/validate-card-string) and [`cardStringsHaveDuplicate`](/docs/reference/api/hands-and-equity/card-strings-have-duplicate) before building larger pipelines.
