# Hand rank labels

`evaluateHandCategory` and `evaluateBestHand().rank` use these labels:

| Label | Order (`handRankCategoryOrder`) |
| --- | --- |
| `highCard` | 0 |
| `onePair` | 1 |
| `twoPair` | 2 |
| `threeOfAKind` | 3 |
| `straight` | 4 |
| `flush` | 5 |
| `fullHouse` | 6 |
| `fourOfAKind` | 7 |
| `straightFlush` | 8 |
| `royalFlush` | 9 |
