# Numerical semantics

RNG choice, parallel seeds, enumeration complexity, and floating-point conventions are documented in the package [`NUMERICAL.md`](https://github.com/DevomB/Poker-Calculations/blob/main/NUMERICAL.md) in the npm repository.

Use reproducible seeds for Monte Carlo and prefer exact APIs when the board has five cards or fewer unknowns.
