# poker-calculations

**poker-calculations** is a Node.js library (N-API + C++20) for NL hold'em: hand evaluation, Monte Carlo and exact equity, pot and rake math, ICM, side pots, draw heuristics, toy GTO-style frequencies, statistics, and more.

## Install

```bash
npm install poker-calculations
```

Requires **Node 18+**. Prebuilt binaries ship for common platforms; see the [npm package](https://www.npmjs.com/package/poker-calculations) and [repository README](https://github.com/DevomB/Poker-Calculations/blob/main/NPM/README.md).

## Quick start

```js
const poker = require('poker-calculations');

console.log(poker.evaluateBestHand(['Ah', 'Kh', 'Qh', 'Jh', 'Th']));
console.log(poker.potOddsRatio(100, 50));
```

## Hand rank labels {#hand-rank-labels}

[`evaluateHandCategory`](/docs/api/hand-evaluation/evaluate-hand-category) returns one of these string labels (lowercase camelCase). Use [`handRankCategoryOrder`](/docs/api/hand-evaluation/hand-rank-category-order) for a numeric sort key `0` … `9`.

| Order | Label |
| --- | --- |
| 0 | `highCard` |
| 1 | `onePair` |
| 2 | `twoPair` |
| 3 | `threeOfAKind` |
| 4 | `straight` |
| 5 | `flush` |
| 6 | `fullHouse` |
| 7 | `fourOfAKind` |
| 8 | `straightFlush` |
| 9 | `royalFlush` |

See also the [package feature inventory](https://github.com/DevomB/Poker-Calculations/blob/main/NPM/FEATURES_ADDED.md#hand-rank-labels) in the repo.
