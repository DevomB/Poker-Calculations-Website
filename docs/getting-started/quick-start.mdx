# Quick start

```js
const poker = require('poker-calculations');

const hand = poker.evaluateBestHand(['Ah', 'Ac', 'Kd', 'Ks', 'Qh']);
console.log(hand.rank);

const equity = poker.simulateHandOutcome(
  ['Ah', 'Kh'],
  ['Qh', 'Jh', 'Th'],
  2000,
  42,
  1,
);

console.log('Equity:', equity);
console.log('SPR:', poker.spr(90, 270));
```

Cards use strings like `"Ah"`, `"Td"` (ten may be `"10h"`). In v2+, hot APIs also accept packed `Uint8Array` deck ids — see [Card notation](/docs/concepts/card-notation).
