# Brand assets (user-provided)

Replace these placeholders before final production polish:

| File | Purpose |
| --- | --- |
| `static/img/logo.svg` | Navbar logo (optional; text title works today) |
| `static/img/favicon.ico` | Browser tab icon |
| `static/img/social-card.png` or `.svg` | Open Graph / Twitter preview |

After adding files, update `favicon` and `themeConfig.image` in `docusaurus.config.ts` if paths change.
